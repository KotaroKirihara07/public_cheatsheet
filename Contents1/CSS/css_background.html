背景

2023/05/10

このページの説明

background：

背景に関する指定をまとめて行う

background: 背景の設定;

background-color：

背景色を指定する

background-color: 色;

background-image：

背景画像を指定する

background-image: url(画像のパス);

#線状グラデーション

background-image: linear-gradient(角度, 開始色, 終了色);

#放射状グラデーション

background-image: radial-gradient(開始色, 終了色);

background-position：

背景画像の表示開始位置を指定する

background-position: 横方向の位置 距離 縦方向の位置 距離;
位置 説明
left 左
center 中央
right 右

background-repeat：

背景画像の繰り返しを指定する

background-repeat: 値;
値 説明
repeat 縦横に繰り返す
no-repeat 繰り返さない
repeat-x 横方向に繰り返す
repeat-y 縦方向に繰り返す

background-size：

背景画像を表示するサイズを指定する

background-size: 値;
値 説明
cover 表示されない部分がある
contain 全体を表示する

background-attachment：

背景画像の位置

background-attachment: 値;
値 説明
scroll スクロール
fixed 固定
