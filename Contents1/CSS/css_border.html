CSS

外枠

2023/05/09

このページの説明

border：

要素の周囲に枠線を付ける

border: 太さ 形状 色;
形状 説明
solid 実線
none 線なし
dotted 点線
dashed 少し長い点線
double 二重線

border-bottom：

要素の底辺に枠線を付ける

border-bottom: 太さ 形状 色;

border-left：

要素の左辺に枠線を付ける

border-left: 太さ 形状 色;

border-radius：

要素の角丸についてまとめて指定する

border-radius: 角丸の半径;

border-right：

要素の右辺に枠線を付ける

border-right: 太さ 形状 色;

border-style：

枠線の種類を指定する

border-style: 太さ 形状 色;

border-top：

要素の上辺に枠線を付ける

border-top: 太さ 形状 色;

border-collapse：

セルとセルの間の罫線を設定する

border-collapse: 値;
値 説明
collapse 罫線は1本
separate 罫線は2本

outline：

要素のボックスに外枠線を設定する

outline: 太さ 形状 色;
