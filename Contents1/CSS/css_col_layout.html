このページの説明

align-items：

flexコンテナ内に含まれる全てのflexアイテムに対して揃える位置を指定する

align-items: ;

flex：

flexbox内のアイテムの伸縮に関する指定をまとめて行う

flex: 拡大率 縮小率 ベースサイズ;

flex-direction：

flexbox内のアイテムの並び順を指定する

flex-direction: ;

flex-wrap：

flexboxに入り切らないアイテムの折り返し方法について指定する

flex-wrap: ;

justify-content：

flexboxコンテナ内でのボックスの詰め方を指定する

justify-content: 値;
値 説明
flex-end 右揃え
center 中央揃え
space-between 均等に配置する
space-around 均等に配置する

order：

フレックスアイテムの並び順を変更する

order: 番号;
