CSS

要素の表示や配置方法

2023/05/10

このページの説明

bottom：

要素を配置する際に、底辺からの距離を指定する

bottom: ;

clear：

要素の回り込みを解除する

clear: 値;
値 説明
left 左フロートを解除する
right 右フロートを解除する
both 両フロートを解除する

display：

要素の表示形式を指定する

display: 表示形式;
表示形式 説明
inline-block 要素をインラインブロックで表示する
block 要素をブロックボックスで表示する
flex

float：

要素を右または左へ寄せて配置する

float: 値;
値 説明
left 左
right 右
none なし

left：

要素を配置する際に、左辺からの距離を指定する

left: 距離;

overflow：

要素からはみ出した内容の表示方法をまとめて指定する

overflow: 表示方法;
表示形式 説明
scroll
hidden フロートを解除する

overflow-x：

要素からはみ出した内容の水平方向の表示方法を指定する

overflow-x: 表示方法;

overflow-y：

要素からはみ出した内容の垂直方向の表示方法を指定する

overflow-y: 表示方法;

position：

要素の配置方法について相対的か絶対的にするかを指定する

position: 値;
表示形式 説明
relative 自由配置したい要素の親要素に指定する
absolute 自由配置したい要素自身に指定する
fixed ビューポートを基準に要素を自由配置する

right：

要素を配置する際に、右辺からの距離を指定する

right: 距離;

vertical-align：

ボックス内のインライン要素に縦位置を揃える基準を指定する

vertical-align: キーワードまたは数値;

visibility：

要素の領域を残したまま表示・非表示を指定する

visibility: ;

z-index：

要素の重なり順序を指定する

z-index: 数値;

opacity：

透明度の設定(0 ~ 1)

opacity: 透明度;

filter：

要素にフィルターを適用する

filter: フィルターの設定;
フィルターの設定 説明
blur(ぼかし量px) 要素をぼかす
brightness(変化率%) 明度を変化させる
contrast(変化率%) コントラストを変化させる
grayscale(変化率%) グレースケールにする
hue-rotate(角度deg) 色相を変化させる
invert(変化率%) 色を反転させる
saturate(変化率%) 彩度を変化させる
sepia(変化率%) セピア色にする

caption-side：

キャプションの位置の設定

caption-side: 位置;
位置 説明
top 上
bottom 下
