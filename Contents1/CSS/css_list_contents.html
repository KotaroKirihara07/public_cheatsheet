CSS

リスト・内容の挿入、置換

2023/05/09

このページの説明

list-style：

リストマーカーに関する指定をまとめて行う

list-style: マークの種類;
マークの種類 説明
disc 「・」デフォルト
circle 白丸
square 四角
none なし
decimal-leading-zero 0で始まる数字
lower-alpha 小文字のアルファベット
upper-alpha 大文字のアルファベット
lower-roman 小文字のローマ数字
upper-roman 大文字のローマ数字
url(/aaa.png) 箇条書きのマークを画像にする

content：

内容を挿入する

content: 内容;

#quoteプロパティで決めた「開始かっこ」を表示する

content: open-quote;
