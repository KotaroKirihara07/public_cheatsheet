カーソルの形状を変更する

カーソルの形状 説明
help ヘルプ
pointer クリックできることを示す
default 通常の矢印ポインタ
wait 処理中で操作できない
not-allowed 操作禁止
zoom-in ズームイン可能
zoom-out ズームアウト可能
