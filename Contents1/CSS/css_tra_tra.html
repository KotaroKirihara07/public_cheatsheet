トランジション・変形

2023/05/09

このページの説明

transition：

時間によって変化するトランジション効果の遷移についてまとめて指定する

transition: 対象のプロパティ 時間 緩急 遅れ;

対象はtransform、opacity

緩急 説明
linear 一定の速度で変化する
ease-in 最初ゆっくりで、だんだん急激に変化する
ease-out 最初急激で、だんだんゆっくり変化する
ease-in-out 最初と最後がゆっくりで、途中急激に変化する
steps(段階) 段階的に変化する。引数は数値
cubic-bezier(x1,y1,x2,y2) ホバーしたときに浮き上がるような効果をつける

#1秒遅れで3秒かけて30度回転する

transform: rotate(30deg); transition: transform 3s liear 1s;

transform：

要素を変形させる

transform: 変形の設定;
変形の設定 説明
rotate(角度deg) 回転
transform(x軸,y軸) 移動
scale(スケール量) 拡大縮小
skew(x軸deg,y軸deg) ゆがめる
