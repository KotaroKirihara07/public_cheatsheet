HTMLの基本的なタグと共通属性

タグ 説明
<!DOCTYPE html> DOCTYPE宣言
<html> ... </html> 文書のルート要素
<head> ... </head> 文書のヘッダであり、ブラウザウインドウには表示されないHTMLドキュメントのメタデータを定義するためのタグ
<meta> 文書に関する情報(メタ情報)
<title> ... </title> 文書のタイトル
<body> ... </body> 文書の本体であり、ブラウザウインドウには表示される部分を配置するためのタグ
<div> ... </div> 範囲の指定(ブロックレベル)
<main> ... </main> ページの中心的なコンテンツ
<section> ... </section> ページ全体の一部分やセクション
<article> ... </article> 記事全体、もしくは独立したコンテンツ
<span> ... </span> 範囲の指定(インライン)
<address> ... </address> 作者情報、連絡先
属性 属性値 説明
class クラス名
id id名

<html> ... </html>

<html>はHTMLのルート要素を配置するためのタグ。

<html 属性="属性値"> ... </html>

属性 属性値 説明
lang ja 日本語
lang en 英語
lang zh 中国語
lang ko 韓国語
lang es スペイン語
lang de ドイツ語
lang fr フランス語

<meta>

<meta>はページのメタ情報を設定するためのタグ。

<meta 属性="属性値">

属性 属性値 説明
charset UTF-8 UTF-8
charset shift_jis Shift JIS
charset euc-jp EUC-JP

#レスポンシブ対応にする

<meta name="viewport" content="width=device-width">

#ページが検索されないようにする

<meta name="robots" content="noindex,nofollow">

#3秒後に別のページに移動する

<meta http-equiv="refresh content="3;url=https://www.-------">

#電話番号がリンクにならないようにする

<meta name="format-detection" content="telephone=no">

<link>

<link>は別ファイルを外部参照するためのタグ。

<link 属性="属性値">

属性 属性値 説明
rel stylesheet CSSファイルの読み込みを可能にする
rel shortcut icon サイトのアイコン(ファビコン)の読み込みを可能にする
href /css/example.css ファイルパスを指定する

#style.cssというcssファイルを参照

<link rel="stylesheet" href="style.css">

#サイトのアイコン(ファビコン)を設定する(設定しなくても"favicon.ico"というファイルをルートディレクトリに置いておけば勝手に読み込まれる)

<link rel="shortcut icon" href="画像ファイルのパス" type="image/vnd.microsoft.icon">

#スマートフォン向けサイトのアイコン(ファビコン)を設定する(ファビコンにする画像の名前をapple-touch-icon.pngとする)

<link rel="apple-touch-icon" href="/apple-touch-icon.png">

#印刷用のCSSを読み込む

<link rel="stylesheet" href="style-print.css" media="print">

<style> ... </style>

<style>はHTMLドキュメント内に直接CSSを記述するためのタグ。

<style> ... </style>

<header> ... </header>

<header>はヘッダーを作成するためのタグ。

<nav> ... </nav>

<nav>はナビゲーションを作成するためのタグ。

#ナビゲーションの作成

<nav class="submenu">
  <ul>
    <li> <a href="#"> aaaaa </a> </li>
    <li> <a href="#"> bbbbb </a> </li>
    <li> <a href="#"> ccccc </a> </li>
  </ul>
</nav>

<script> ... </script>

<script>はJavaScriptファイルを読み込むためのタグ。

<script 属性="属性値"> ... </script>

#script.jsというファイルを読み込む

<script src="/script.js"> </script>

#HTMLに直接JavaScriptを埋め込む

<script> JavaScriptプログラム </script>

#JavaScriptファイルを非同期で読み込み、即座に実行

<script src="/script.js" async> </script>

#JavaScriptファイルを非同期で読み込み、HTMLの解析完了後に実行

<script src="/script.js" defer> </script>

#ESモジュール(処理ごとに分割したJavaScriptファイルを読み込み実行する) (script.jsからMyclass1.jsやMyClass2.jsを呼び出す)

<script type="module" src="/script.js"> </script>

<noscript> ... </noscript>

<noscript>はJavaScriptが動作しないブラウザが表示するコンテンツを記述するためのタグ。

<noscript> ... </noscript>

#JavaScriptが動作しないブラウザが表示するコンテンツを記述する

<noscript> 表示するコンテンツ(HTML) </noscript>

<footer> ... </footer>

<footer>はフッターを作成するためのタグ。
