フォーム

2024/06/06

このページではフォームの説明を行う。

<form> ... </form>

<form>はフォームを作成するためのタグ。

<form 属性="属性値"> ... </form>

属性 属性値 説明
action URL or Path 宛先URLまたは宛先パスを設定する
method GET URLの一部分に入力した内容を足して送信する方法(個人情報の送信には適さない)
method POST URLを変化させることなく送信する方法

#formテンプレート

<form action="http:----" method="POST"> ... </form>

<input>

<input>は入力フォームを表すタグ。

<input 属性="属性値"/>

属性 属性値 説明
type button ボタン
type checkbox チェックボックス
type email メール
type file ファイル
type number 数字
type radio ラジオボタン
type password パスワード
type reset リセット
type submit 送信
type tel 電話番号
type text テキスト
type url URL
type img 画像
placeholder ヒントを表示 入力のヒントを表示する
value 数値やテキスト 送信する値を設定する
name 名前 パーツに名前を付与する
autofocus - ページが表示されたとき、自動でこのフォーム部品が入力可能になる
required - 必須入力項目にする
readonly - 入力はできないが選択のみできるようにする (value属性とセットで使用する)

#個人情報入力フォームの作成

<form action="http:----" method="POST">
   <p> 名前　　　　　　: <input type="text" name="name" /> </p>
   <p> 生年月日　　　　: <input type="date" name="birthday" /> </p>
   <p> パワー　　　　　: <input type="number" name="power" /> </p>
   <p> メールアドレス　: <input type="email" name="email_form" /> </p>
   <p> 電話番号　　　　: <input type="tel" name="tel" /> </p>
   <p> パスワード　　　: <input type="password" name="pass" /> </p>
   <p> テキスト　　　　: <input type="text" name="text" /> </p>
   <input type="reset" value="リセット" />
   <input type="submit" value="送信" />
</form>

#複数の選択肢から一つ選ばせる (ラジオボタン)

<form action="http:----" method="POST">
   <input type="radio" name="rdo" value="1" /> AAAAA
   <input type="radio" name="rdo" value="2" /> BBBBB
   <input type="radio" name="rdo" value="3" /> CCCCC
   <br>
   <input type="reset" value="リセット" />
   <input type="submit" value="送信" />
</form>

#複数の選択肢から複数選ばせる (チェックボックス)

<form action="http:----" method="POST">
   <input type="checkbox" name="check" value="A" /> AAAAA
   <input type="checkbox" name="check" value="B" /> BBBBB
   <input type="checkbox" name="check" value="C" /> CCCCC
   <br>
   <input type="reset" value="リセット" />
   <input type="submit" value="送信" />
</form>

#ファイルをアップロードする

<form action="http:----" method="POST">
   <input type="file" name="file" />
   <br> <br>
   <input type="reset" value="リセット" />
   <input type="submit" value="送信" />
</form>

#非表示データを埋め込む

<form action="http:----" method="POST">
   <input type="hidden" name="hdd" value="hiddenvalue" />
</form>

#送信ボタンを画像にする

<form action="http:----" method="POST">
   <p> テキスト : <input type="text" name="text" /> </p>
   <input type="image" src="image.png" alt="申し込む" />
</form>

#画面遷移するためのボタン

<input type="button" onclick="location.href='http://----'" value="画面遷移ボタン" />

<textarea> </textarea>

<textarea>は複数行のテキストエリアを作成するためのタグ

<textarea name="パーツ名" cols="幅" rows="行数" placefolder="初期値"> </textarea>

#複数行のテキストエリアを作成

<textarea name="message" id="message" rows="10" placefolder="感想を入力してください。"> </textarea>

#boot-strapで複数行のテキストエリアを作成

<textarea name="message" id="message" rows="10" class="form-control"> </textarea>

<select> ... </select>

<select>はセレクトボックス作成のためのタグ。<optgroup>と<option>をセットで使用する。

<select name="パーツ名">
  <optgroup 属性="属性値">
    <option value="値"> ... </option>
    <option value="値"> ... </option>
    <option value="値"> ... </option>
  </optgroup>
</select>

#boot-strapで入力フォームの作成

<select id="job" name="job" class="form-control">

#職業を選択するためのドロップダウンリスト(1つだけ選択できる)

<select name="job" id="job">
  <option value="" selected> 選択してください </option>
  <option value="会社員"> 会社員 </option>
  <option value="学生"> 学生 </option>
  <option value="無職"> 無職 </option>
</select>

#行ったことのある場所のドロップダウンリスト(複数選択できる)

<select name="place" id="place" multiple>
  <option value="" selected> 選択してください </option>
  <option value="東京"> 東京 </option>
  <option value="大阪"> 大阪 </option>
  <option value="福岡"> 福岡 </option>
</select>

#選択肢をグループ化したドロップダウンリスト(1つだけ選択できる)

<select name="place" id="place">
  <option value="" selected> 選択してください </option>
  <optgroup label="関東">
    <option value="東京"> 東京 </option>
    <option value="神奈川"> 神奈川 </option>
  </optgroup>
  <optgroup label="関西">
    <option value="大阪"> 大阪 </option>
    <option value="京都"> 京都 </option>
  </optgroup>
</select>

<button> ... </button>

<button>はボタンを作成するためのタグ。

<button 属性名="属性値"> ボタン </button>

属性 属性値 内容
type submit 送信ボタン
type reset リセットボタン

#「登録する」ボタンをつくる

<button type="submit"> 登録する </button>

<label> ... </label>

<label>はラベルを設定するためのタグ。

<label 属性="属性値"> ... </label>

属性 属性値 説明
for id 関連するフォーム部品のid属性と接続する

#メールアドレスのラベルの作成

<label for="mymail"> メールアドレス </label>

<fieldset> ... </fieldset>

<fieldset>はフォーム部品のグループ化をするためのタグ(<legend>とセットで使用する)

<fieldset> ... </fieldset>

#フォーム部品のグループ化をする

<fieldset>
  <legend> 基礎データを入力してください </legend>
  <p> <label> 体重 <input type="text" name="weight"> </label> kg </p>
  <p> <label> 身長 <input type="text" name="height"> </label> cm </p>
</fieldset>
