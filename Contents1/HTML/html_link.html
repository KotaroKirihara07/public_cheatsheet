HTML

リンク

2024/06/16

このページのではリンクに関する説明を行う。

<a> ... </a>

<a>はリンクを作成するためのタグ。

<a 属性="属性値"> ... </a>

属性 属性値 説明
href Path or URL 該当PathまたはURLのページに移動
href / ホームに戻る
href #id ページ内の特定のidをもったリンクに飛ぶ
href Path#id Pathページ内の特定のidをもったリンクに飛ぶ
href mailto:contact@example.com メールアドレスをリンクにする
href tel:12345678900 電話番号をリンクにする
href /sample.pdf ダウンロードできるファイルのパス (download属性とセットで使用)
target _blank 新しいウィンドウで開く
target _self 自分自身のウィンドウで開く
target _top フレームを解除して開く
target _parent 親フレームで開く
download sample.pdf ダウンロードする際のファイル名の設定 (href属性とセットで使用)
title テキスト ツールチップの表示
style text-decoration:none; 下線を非表示にする

<video> ... </video>

<video>はWebサーバにアップロードした動画ファイルを表示するためのタグ。

<video 属性="属性値"> ... </video>

属性 属性値 説明
src URL or Path 動画のパスorURLの設置
poster URL or Path 動画が再生されないときに表示する静止画の設定
width 数値 横幅の設定
height 数値 縦幅の設定
autoplay - 自動再生オン
controls - 再生コントローラを表示する
loop - ループ再生する
muted - 消音する
playsinline - ページ内で再生
