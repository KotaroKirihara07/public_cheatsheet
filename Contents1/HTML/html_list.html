リスト

2024/02/03

このページではリストに関する説明を行う。

<ul> ... </ul>

<ul>は順不同のリストを作成するためのタグ。

<ul>
  <li> ... </li>
  <li> ... </li>
  <li> ... </li>
</ul>

<ol> ... </ol>

<ol>は順序つきのリストを作成するためのタグ。

<ol 属性="属性値"> ... </ol>

<ol>
  <li> ... </li>
  <li> ... </li>
  <li> ... </li>
</ol>

属性 属性値 説明
start 数値 開始番号を指定する

<dl> ... </dl>

<dl>は定義型リストを作成するためのタグ。

<dl>
  <dt> キーワード1 </dt>
  <dd> 定義1 </dd>
  <dt> キーワード2 </dt>
  <dd> 定義2 </dd>
  <dt> キーワード3 </dt>
  <dd> 定義3 </dd>
</dl>
