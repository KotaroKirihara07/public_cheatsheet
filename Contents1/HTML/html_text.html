テキスト表現

2024/02/03

このページではテキスト表現に関する説明を行う。

<h1> ... </h1>

<h1>は見出しを作成するためのタグ。

#見出しを作成

<h1> example text </h1>

<p> ... </p>

<p>はパラグラフを作成するためのタグ。

#パラグラフを作成

<p> example text </p>

<br>

<br>は改行を行うためのタグ。

#改行

<br>

<hr>

<hr>は水平線を表示するためのタグ。

#水平線を表示する

<hr>

テキスト装飾

テキスト装飾に関するタグについて以下の表にまとめる。

タグ 出力 説明
<big> ... </big> example text 大きめの文字
<small> ... </small> example text 小さめの文字
<b> ... </b> example text 太字
<u> ... </u> example text 下線
<i> ... </i> example text イタリック体
<em> ... </em> example text 強調
<strong> ... </strong> example text より強い強調
<mark> ... </mark> example text マーカーで強調
<del> ... </del> example text 削除されたテキスト
<ins> ... </ins> example text 追加されたテキスト
<sup> ... </sup> example text 上付き文字
<sub> ... </sub> example text 下付き文字

<abbr> ... </abbr>

<abbr>は略語の正式名称を表示するためのタグ。

<abbr title="正式名称"> 略語 </abbr>

#ICT (Information and Communication Technology)を表示する

<abbr title="Information and Communication Technology"> ICT </abbr>

<q> ... </q>、<blockquote> ... </blockquote>、<cite> ... </cite>

<q>,<blockquote>,<cite>は引用を行うためのタグ。

#短文の引用(インライン)

<q> example text </q>

#引用元URLとともに引用した部分を表示(インライン)

<q cite="https://www.example.com"> example text </q>

#長文の引用(ブロックレベル)

<blockquote> example text </blockquote>

#引用元URLとともに引用した部分を表示(ブロックレベル)

<blockquote cite="https://www.example.com"> example text </blockquote>

#引用元(出典、参照先)

<cite> example text </cite>

<code> ... </code>、<pre> ... </pre>

<code>、<pre>はソースコードを表示するためのタグ。

#一行のソースコードを表示

<code> print(777) </code>

#複数行のソースコードを表示

<pre> print(777) </pre>

<time> ... </time>

<time>は日付や時間を表示するためのタグ。

属性 属性値 説明
datetime 2020-01-01 20201年1月1日
datetime 2020-01-01T16:30 20201年1月1日16時30分
datetime 2020-01-01T16:30-0900 20201年1月1日16時30分 (日本時間)

<ruby> ... </ruby>、<rb> ... </rb>、<rp> ... </rp>、<rt> ... </rt>

<ruby>、<rb>、<rp>、<rt>は漢字にルビを振るためのタグ。

#ルビを振る

<ruby> <rb> 紫陽花 </rb> <rp>(</rp> <rt> あじさい </rt> <rp>)</rp> </ruby>

<kbd> ... </kbd>

<kbd>はキーボードのキーを表示するためのタグ。

#Ctrl + Cを表示する

<kbd> Ctrl </kbd> + <kbd> C </kbd>
