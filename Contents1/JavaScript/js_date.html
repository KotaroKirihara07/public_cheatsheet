Dateオブジェクト

2024/07/20

このページでは日付や時間の操作方法についての説明を行う。

日付や時間の操作方法

#インスタンス化 (現在時刻)

let date = new Date();

#インスタンス化 (特定の日時)

let date1 = new Date('2020/01/01/ 13:00:00');
let date2 = new Date(2020,1,1,13,0,0);

#静的メソッド集

メソッド 説明 戻り値の型
Date.parse('2020/01/01') 日付文字列からタイムスタンプ値を取得する(1970/01/01から?ミリ秒経過したか) 数値
Date.now() 現在時刻のタイムスタンプ値を取得する 数値

#日付や時間を取得するためのメソッド集

メソッド 説明 戻り値の型
getFullYear() 現在の西暦を取得する 数値
getMonth() 現在の月を取得する (0が1月) 数値
getDate() 現在の日付を取得する 数値
getHours() 現在の時間を取得する 数値
getMinutes() 現在の分数を取得する 数値
getSeconds() 現在の秒数を取得する 数値
getMilliseconds() 現在のミリ秒数を取得する 数値
getDay() 現在の曜日を取得する (0が日曜日) 数値
toLocaleDateString() 日付の文字列を取得する 文字列
toLocaleTimeString() 時刻の文字列を取得する 文字列

#日付や時間を設定するためのメソッド集

メソッド 説明 戻り値の型
setFullYear(2020) 西暦を設定 数値
setMonth(7) 月を設定 数値
setDate(7) 日を設定 数値
setHours(14) 時を設定 数値
setFullYear(32) 分を設定 数値
setFullYear(50) 秒を設定 数値
setFullYear(300) ミリ秒を設定 数値
setFullYear(2020) 西暦を設定 数値
