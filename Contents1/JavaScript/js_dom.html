このページではHTML要素の操作方法について説明を行う。

HTML要素の取得

#セレクタ名に一致する要素を取得する

const element = document.querySelector('.class_name');
const element = document.querySelector('#id_name');

#IDに一致する要素を取得する

const element = document.getElementById('id_name');

#セレクタ名に一致するすべての要素を取得する

const element = document.querySelectorAll('.class_name');

#クラス名に一致するすべての要素を取得する

const element = document.getElementByClassName('class_name');

HTMLの要素の操作

要素の取得

#プロパティ集

プロパティ 意味 型
document.documentElement ルート要素 html要素
document.head head要素 head要素
document.body body要素 body要素

親、子、前後要素の取得

#プロパティ集

プロパティ 意味 型
親ノード.children 子ノード 要素群
親ノード.firstElementChild 最初の子ノード 要素
親ノード.lastElementChild 最後の子ノード 要素
ノード.nextElementSibling 次(弟)のノード 要素
ノード.previousElementSibling 前(兄)のノード 要素
子ノード.parentNode 親のノード ノード

要素の追加・削除

#メソッド集

メソッド 意味 戻り値
親ノード.appendChild(子ノード) 親ノードの末尾に子ノードを追加する 要素群
親ノード.insertBefore(子ノード,指定ノード) 親ノード内の指定ノード前に子ノードを追加する 要素
ノード1.before(ノード2) ノード1の前にノード2を追加する なし
ノード1.after(ノード2) ノード1の後にノード2を追加する なし
親ノード.hasChild(子ノード) 親ノードに子ノードが存在するかどうか 真偽値
親ノード.removeChild(子ノード) 親要素の子要素を取り除く 要素
ノード.remove() 要素を取り除く なし
ノード.cloneNode() 要素を複製する ノード
親ノード.replaceChild(新ノード,旧ノード) 新ノードと子ノードを置き換える 置き換えられたノード
旧ノード.replaceWith(新ノード) 旧ノードと新ノードで置き換える なし
ノード.textContent ノード内のテキスト 文字列

#HTMLコードを要素として挿入する

const newelement = 'HTML';
親要素.insertAdjacentHTML(挿入位置,newelement);

挿入位置 意味
'beforebegin' 親要素の直前
'afterbegin' 親要素内の先頭
'beforeend' 親要素内の末尾
'afterend' 親要素の直後

要素の生成

const divelement = document.createElement('タグ名');

要素の操作


#プロパティ集

プロパティ 意味 型
divelement.innerHTML 要素内のHTML文字列 文字列
divelement.outerHTML 要素自身のHTML文字列 文字列
divelement.style.プロパティ名 スタイルの値 文字列
getComputedStyle(divelement).プロパティ名 スタイルの値 文字列

#メソッド集

メソッド 意味 型
divelement.setAttribute(属性名,値) 要素の属性を設定する なし
divelement.getAttribute(属性名) 要素の属性を取得する なし
divelement.hasAttribute(属性名) 要素の属性があるかどうか 真偽値
divelement.classList.add(クラス1,クラス2,...) クラスを追加する なし
divelement.classList.remove(クラス1,クラス2,...) クラスを削除する なし
divelement.classList.contains(クラス) クラスが存在するか 真偽値
divelement.classList.toggle(クラス) クラスを切り替える なし

formの情報取得

#formに入力された情報を取得する

const element = document.querySelector('#id_name');
const value = element.value;

#チェックボックスの情報を取得する (true or false)

const element = document.querySelector('#id_name');
const check = element.checked;

#ローカルファイルの情報を取得する

const element = document.querySelector('#file');
const target = element.target;
const files = target.files;
const file = files[0];
