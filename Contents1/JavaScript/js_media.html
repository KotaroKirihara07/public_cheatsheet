メディアの操作

2024/02/09

メディア (アニメーション、画像、音声、動画)の操作方法について説明を行う。

アニメーション

#モーションを作成する

const element = document.querySelector('.class_name'); element.animate(
  {
    transform: [
      //開始値
      処理
      //終了値
      処理
    ]
  },
  {
    //ミリ秒指定
    duration: 3000,
    //繰り返し回数
    iterations: Infinity,
    //繰り返し挙動
    direction: 'normal',
    //加減速種類
    easing: 'ease',
    //終了時のプロパティ
    fill: 'forwards',
  }
);

画像

#画像をスクリプトで読み込む

const img = document.querySelector('#id_name');
img.src = 'images/sample.jpg';

#スクリプトからimg要素を追加する

const img = new Image();
img.src = 'images/sample.jpg';

音声

#音声をスクリプトで制御する

const audio = document.querySelector('#id_name');
audio.play();
audio.pause();

#プロパティ集

プロパティ 説明
audio.currentTime 現在の秒数を取得する (or 変更する)
audio.duration 音声の長さ(秒数)を取得する
audio.volume 音声のvolumeを取得する (or 変更する)
audio.muted = true ミュートオン

動画

#動画をスクリプトで制御する

const video = document.querySelector('#id_name');
video.play();
video.pause();
