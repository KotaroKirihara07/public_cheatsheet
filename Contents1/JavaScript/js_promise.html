Promiseオブジェクト

2024/07/21

このページでは処理タイミングの制御方法について説明を行う。

非同期処理

#インスタンス化

const promise = new Promise((resolve,reject) => {
  resolve();
  reject();
});

#成功したときのコールバック関数の呼び出し

promise.then( () => {
  処理
});

#失敗したときのコールバック関数の呼び出し

promise.catch( () => {
  処理
});

#直列処理

Promise.resolve()
  .then(
    () =>
      new Promise((resolve) => {
        処理
      })
    )
  .then(
    () =>
      new Promise((resolve) => {
        処理
      })
    );
