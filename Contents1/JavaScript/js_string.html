Stringオブジェクト

2024/07/21

このページでは文字列について説明を行う。

文字列

#文字列を定義する

let str1 = 'Hello!'
const str2 = 'World!'

#文字列の中で変数を扱う

let str3 = '私は${value}歳です。'

#文字列を結合させる (その1)

str4 = str1 + str2 + str3 + ...

#文字列を結合させる (その2)

str4 = '${str1}${str2}${str3}'

#メソッド集

メソッド 説明 戻り値
str.length 文字列の長さを取得する 数値
str.trim() 文字列の両端の空白を取り除く 文字列
str.indexOf(検索文字列) 検索文字列が最初に現れるインデックス 数値
str.lastindexOf(検索文字列) 検索文字列が最後に現れるインデックス 数値
str.search(正規表現) 正規表現にマッチする文字列のインデックス 数値
str.includes(検索文字列) 検索文字列が含まれているか 真偽値
str.startWith(検索文字列) 検索文字列で始まるかどうか 真偽値
str.endWith(検索文字列) 検索文字列で終わるかどうか 真偽値
str.charAt(インデックス) 指定インデックスの文字を取得する 文字
str.slice(開始インデックス,終了インデックス) 指定範囲内の文字列を取得する 文字列
str.substring(開始インデックス,終了インデックス) 指定範囲内の文字列を取得する 文字列
str.substr(開始インデックス,文字数) 指定範囲内の文字列を取得する 文字列
str.replace(文字列1,文字列2) 文字列1を文字列2に置き換える 文字列
str.split(区切り文字) 区切り文字で分割したリストを返す リスト
str.toLowerCase() 文字列を小文字に変換する 文字列
str.toUpperCase() 文字列を大文字に変換する 文字列
