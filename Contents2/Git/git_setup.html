Git

セットアップ

2024/11/14

このページではGitのセットアップ方法について説明する。

インストール

Linux (Redhat)

sudo yum install git-all

Linux (Debian)

sudo apt-get install git-all

Windows

  1. https://git-scm.com/downloadsで該当OSを選択する
  2. "Click here to download"をクリックしてインストール

GitHubとSSH通信をするための設定

説明

  1. cd ~/.ssh
  2. ssh-keygen -t rsa
  3. 1つ目の質問を id_github_rsaとしてEnter
  4. GitHubにログイン
  5. 「Add SSH key」のボタンを押す
  6. titleを「id_github_rsa」にする
  7. Keyは「id_github_rsa.pub」の中身をコピペする
  8. ~/.ssh/configを作成する
  9. 「config」ファイルに「g_config」ファイルの内容をコピペ (Templateディレクトリを参照)
  10. ~/.gitconfigを作成
  11. 「.gitconfig」ファイルに「git_config」ファイルの内容をコピペ (Templateディレクトリを参照)
  12. ssh -T githubと入力して接続を確認

GitlabとSSH通信をするための設定

説明

  1. cd ~/.ssh
  2. ssh-keygen -t rsa -C アドレス@example.com
  3. 1つ目の質問を id_gitlab_rsaとしてEnter
  4. GitHubにログイン
  5. 「Add SSH key」のボタンを押す
  6. titleを「id_gitlab_rsa」にする
  7. Keyは「id_gitlab_rsa.pub」の中身をコピペする
  8. ~/.ssh/configを作成する
  9. 「config」ファイルに「g_config」ファイルの内容をコピペ (Templateディレクトリを参照)
