CASE式

2024/02/16

このページではCASE式についての説明を行う。CASE式は「検索CASE式」と「単純CASE式」の2種類存在する。CASE式は条件によって処理を分岐させるために使用する。

CASE式

条件によって処理を分岐させる。

検索CASE式

SELECT col_name1, col_name2,
  CASE
    WHEN col_name3 = 'a' THEN 処理1
    WHEN col_name3 = 'b' THEN 処理2
    WHEN col_name3 = 'c' THEN 処理3
    ELSE 処理4
  END AS new_col_name3
FROM Table_name;

単純CASE式

SELECT col_name1, col_name2,
  CASE col_name3
    WHEN 'a' THEN 処理1
    WHEN 'b' THEN 処理2
    WHEN 'c' THEN 処理3
    ELSE 処理4
  END AS new_col_name3
FROM Table_name;
