述語

2024/02/16

このページでは述語にについての説明を行う。述語とは、戻り値がTURE/FALSE/UNKWONNのいずれかである関数である。

LIKE述語

文字列の部分一致検索。「%」は0文字以上の任意の文字列、「_」は任意の1文字を表す。

#前方一致検索

SELECT * FROM Table_name WHERE col_name LIKE 'abc%';

#中間一致検索

SELECT * FROM Table_name WHERE col_name LIKE '%bcd%';

#後方一致検索

SELECT * FROM Table_name WHERE col_name LIKE '%cde';

#特定の文字数の後方一致検索(前2文字は任意)

SELECT * FROM Table_name WHERE col_name LIKE '__cde';

BETWEEN述語

範囲検索。

#col_name2の値が100以上200以下の行を取得する

SELECT col_name1, col_name2 FROM Table_name WHERE col_name2 BETWEEN 100 AND 200;

IS NULL述語

値がNULLの行の取得。または、値がNULLでないの行の取得。

#値がNULLの行の取得

SELECT * FROM Table_name WHERE col_name1 IS NULL;

#値がNULLでない行の取得

SELECT * FROM Table_name WHERE col_name1 IS NOT NULL;

IN述語

条件を複数同時に与えることを可能にする。

#col_name2の値が(100,200,300)のいずれかの場合の行を取り出す

SELECT col_name1,col_name2 FROM Table_name WHERE col_name2 IN (100,200,300);

#col_name2の値が(100,200,300)のいずれでもない場合の行を取り出す

SELECT col_name1,col_name2 FROM Table_name WHERE col_name2 NOT IN (100,200,300);

EXISTS述語

IN述語で代用できるため積極的に覚える必要はなし。必要になったら勉強してください。
