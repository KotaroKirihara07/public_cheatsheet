MVC

Model ： データアクセス関係の処理を担当。Webアプリケーションとデータベース間のやりとりを行う。

View ： 画面に表示されるWebページの作成を担当。

Controller ： 全体の制御を担当。

トップページの作成

説明

  1. プロジェクト名と同じ名前のディレクトリ内の「urls.py」を開く
  2. from djapp import viewsとpath('', views.index, name='index')を追加する
  3. djappディレクトリに「views.py」を作成する
  4. 「views.py」を 

    from django.shortcuts import render

def index(request):
    return render(request, 'djapp/index.html')
    とする
  5. 上層のdjappディレクトリ内に「templates」ディレクトリを作成する
  6. 「templates」ディレクトリに「djapp」ディレクトリを作成する
  7. 「djapp」ディレクトリに「index.html」を作成する
  8. 下層のdjappの「setting.py」にimport osを追加した後、[TEMPLATES]内の[DIRS]を[os.path.join(BASE_DIR, 'templates')]にする

アプリケーション開発の流れ

  1. プロジェクトを立ち上げる
  2. アプリケーションを立ち上げる
  3. 「setting.py」を編集し、アプリケーションを追加する
  4. 「models.py」を編集し、データベースを定義する
  5. マイグレーションファイルを作成し、マイグレーションを実行する
  6. 「admin.py」を編集し、データベースを管理可能な状態にする
  7. 「templatesディレクトリ」にhtmlファイルを配置する
  8. 「views.py」で関数を定義する
  9. 「urls.py」を編集する
  10. 実行
