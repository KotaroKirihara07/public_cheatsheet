データベースの管理

2024/08/06

このページではデータベース管理者についての説明を行う。

管理者の作成

  1. python manage.py createsuperuser
  2. 管理者名を「admin」とする
  3. メールアドレスを入力する
  4. パスワード(8文字以上)を入力する
  5. パスワードを再入力する

admin.pyの編集

「admin.py」を編集することでadminが管理することができるモデルを追加することができる。以下で扱うモデル名は「model.py」で定義したモデル名である。

  1. from .models import モデル名 を追加
  2. admin.site.register(モデル名) を追加

#以下テンプレート

from django.contrib import admin
from .models import model_name

admin.site.register(model_name)

管理者ツールにログインする

  1. ブラウザで http://localhost:8000/admin と入力する
  2. ユーザ名とパスワードを入力する

利用者を追加する

  1. 「ADD USER」をクリックする
  2. ユーザ名を入力する
  3. パスワード(8文字以上)を入力する
  4. パスワードを再入力する
