Django

models.py

2023/06/25

models.pyはモデルを定義するためのファイル。モデルを定義することでデータベースも自動的に生成される。

モデルの作成

「model.py」を編集する。 

class model_name(models.Model):
    項目1 = フィールドのインスタンス1
    項目2 = フィールドのインスタンス2
    項目3 = フィールドのインスタンス3

#フィールドのインスタンス

フィールドのインスタンス 説明
models.CharField(max_length=max) 文字列
models.IntegerField(default=0,max_length=max) 整数
models.BooleanField() 真偽値
models.EmailField(max_length=max) メール
models.DateField() 日付

バリデーション

#CharField (文字数の制限)

from django.core.validators import MinLengthValidator,MaxLengthValidator
models.CharField(validators=[MinLengthValidator(0),MaxLengthValidator(10))

#IntegerField (最小値と最大値の設定)

from django.core.validators import MinValueValidator,MaxValueValidator
models.IntegerField(validators=[MinValueValidator(0),MaxValueValidator(10)])

#ProhibitNullCharactersvalidator (null文字の禁止)

from django.core.validators import ProhibitNullCharactersvalidator
validators=[ProhibitNullCharactersvalidator()]

#RegexValidator (正規表現パターン)

from django.core.validators import RegexValidator
validators=[RegexValidator(正規表現)]

リレーションシップ

#1対1対応

1つの「主モデル」に対して1つの「従モデル」を関連づける方式。「主モデル」は特に変更する必要はなく、「従モデル」において以下を追加する。

項目 = models.OneToOneField(主モデル名)

#1対多対応

1つの「主モデル」に対して複数の「従モデル」を関連づける方式。「主モデル」は特に変更する必要はなく、「従モデル」において以下を追加する。

項目 = models.ForeignKey(主モデル名)

#多対多対応

複数の「主モデル」に対して複数の「従モデル」を関連づける方式。「主モデル」は特に変更する必要はなく、「従モデル」において以下を追加する。

項目 = models.ManyToManyField(主モデル名)

テンプレート 

from django.db import models

# Create your models here.
class model_name(models.Model):
    name = models.CharField(max_length=10)
    mail = models.EmailField(max_length=20)
    gender = models.BooleanField()
    age = models.IntegerField(default=0)
    birthday = models.DateField()
              
    def __str__(self):
        return "<model_name:id = " +str(self.id) + "," + self.name + "(" + str(self.age) + ")>"
