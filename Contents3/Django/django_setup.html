Django

セットアップ

2024/07/28

このページではDjangoのセットアップ方法について説明する。

インストール

Pythonをインストールした状態で以下のコードを実行

python -m pip install Django

プロジェクトディレクトリの構成

プロジェクトディレクトリとは、プロジェクトと同名のディレクトリである。

ファイル名 内容
__init__.py Djangoプロジェクトを実行するときの初期化処理を行うスクリプトファイル
asgi.py ?
setting.py プロジェクトの設定情報を記述したファイル
urls.py プロジェクトで使うURLを管理するファイル
wsgi.py Webアプリケーションのメインプログラムとなる部分

アプリケーションディレクトリの構成

アプリケーションディレクトリとは、作成した任意のアプリケーションと同名のディレクトリである。

ディレクトリ/ファイル名 内容
migrations データベース関係の機能のファイルがまとめられているディレクトリ
__init__.py アプリケーションの初期化処理のためのファイル
admin.py 管理者ツールのためのファイル
apps.py アプリケーション本体の処理のためのファイル
models.py モデルに関する処理を記述するためのファイル
tests.py プログラムのテストのためのファイル
views.py 画面表示のためのファイル
