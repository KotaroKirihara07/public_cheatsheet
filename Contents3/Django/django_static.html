staticディレクトリ

2024/07/29

このページではcssやjavascriptや画像を扱う方法について説明する。

準備

  1. アプリケーションディレクトリ内に「static」という名前のディレクトリを作成する
  2. 「static」という名前のディレクトリ内にアプリケーション名と同じ名前のディレクトリを作成する
  3. アプリケーション名と同じ名前のディレクトリ直下に「css」という名前のディレクトリを作成する
  4. アプリケーション名と同じ名前のディレクトリ直下に「js」という名前のディレクトリを作成する
  5. アプリケーション名と同じ名前のディレクトリ直下に「img」という名前のディレクトリを作成する

cssファイルを適用させる方法

  1. htmlファイルの先頭に {% load static %} を追加する
  2. <head>内に <link reL="stylesheet" type="text/css" href={% static 'app_name/css/style.css'%}> を追加する

javascriptファイルを適用させる方法

  1. htmlファイルの先頭に {% load static %} を追加する
  2. <head>内に <script src="{% static 'app_name/js/sample.js' %}"> </script>を追加する

画像ファイルを読み込む方法

  1. htmlファイルの先頭に {% load static %} を追加する
  2. 任意の場所に <img src="{% static "app_name/img/sample.png" %}" alt="表示できません"> を追加する
