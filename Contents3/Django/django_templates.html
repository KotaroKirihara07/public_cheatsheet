tempaltesディレクトリはHTMLファイルを配置するためのディレクトリである。ここに配置されるHTMLファイルの書式が通常のHTMLの書式と異なる箇所があるため注意が必要である。

準備

  1. アプリケーションディレクトリ(app_name)内に「templates」ディレクトリを作成する
  2. 「templates」ディレクトリ内にアプリケーション名と同じ名前のディレクトリを作成する
  3. アプリケーション名と同じ名前のディレクトリ内に「index.html」を作成する

HTMLファイルの編集

#コメント

{% comment %} コメント {% endcomment %}

#「views.py」のparamsで指定した変数を扱う

{{title}}, {{message}}, {{data}}

#他のページへのリンク ( 「urls.py」の「urlpatterns」のpathのname(=path_name)の値を名前にする )

<a href="{% url path_name %}"> リンク先 </a>

#for文

{% for item in data %}
    #処理
{% endfor %}

#if文

{% if item.id == 1%}
    #処理
{% endif %}

#ジェネリックビューで取り出したレコードのリスト

{{object_list}}

フォーム

#HTMLのタグを直接書いてフォームを作成 (送信先は「view.py」ファイル内のurlpatternsのname='form'というpath)

<form action="{% url 'form' %}" method="POST">
    {% csrf_token %}
    <label for="msg"> message </label>
    <input id="msg" type="text" name="msg">
    <input type="submit" value="送信">
</form>

#フォームの自動生成

<form action="{% url 'form' %}" method="POST">
    {% csrf_token %}
    {{ form }}
    <input type="submit" value="送信">
</form>

#テーブル型のフォームの自動生成

<form action="{% url 'form' %}" method="POST">
    {% csrf_token %}
    {{ form.as_table }}
    <input type="submit" value="送信">
</form>

#リスト型のフォームの自動生成

<form action="{% url 'form' %}" method="POST">
    {% csrf_token %}
    {{ form.as_ul }}
    <input type="submit" value="送信">
</form>

#パラグラフ型のフォームの自動生成

<form action="{% url 'form' %}" method="POST">
    {% csrf_token %}
    {{ form.as_p }}
    <input type="submit" value="送信">
</form>

ページネーション

#ページ移動用のリンク

<ul class="pagination justify-content-center">
    {% if data.has_previous %}
    <li class="page-item">
        <a class="page-link" href="{% url 'page' %}"> « first </a>
    </li>
    <li class="page-item">
        <a class="page-link" href="{% url 'page' %}{{data.previous_page_number}}"> « prev </a>
    </li>
    {% else %}
    <li class="page-item">
        <a class="page-link"> « first </a>
    </li>
    <li class="page-item">
        <a class="page-link"> « prev </a>
    </li>
    {% endif %}

    <li class="page-item">
        <a class="page-link"> {{data.number}}/{{data.paginator.num_pages}} » </a>
    </li>

    {% if data.has_next %}
    <li class="page-item">
        <a class="page-link" href="{% url 'page' %}{{data.next_page_number}}"> next » </a>
    </li>
    <li class="page-item">
        <a class="page-link" href="{% url 'page' %}{{data.paginator.num_pages}}"> last » </a>
    </li>
    {% else %}
    <li class="page-item">
        <a class="page-link"> next » </a>
    </li>
    <li class="page-item">
        <a class="page-link"> last » </a>
    </li>
    {% endif %}
</ul>

レイアウト用テンプレート

#layout用のHTMLファイル

{% block title %} {% endblock %}
{% block header %} {% endblock %}
{% block content %} {% endblock %}

#テンプレート用HTMLファイル

  1. {% extends 'app_name/layout.html' %} を先頭に追加する
  2. {% block title %} と {% endblock %} の間にタイトルを記述する
  3. {% block header %} と {% endblock %} の間にヘッダーを記述する
  4. {% block content %} と {% endblock %} の間にコンテンツを記述する
