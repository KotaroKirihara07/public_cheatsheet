urls.py

2024/08/04

このページではurls.pyの編集方法について説明する。

プロジェクトディレクトリの「urls.py」

#編集方法

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    #新規アプリケーションのpathの追加
    path('app_name/', include('app_name.urls')),
]

アプリケーションディレクトリの「urls.py」

基本的にはアプリケーションごとの「urls.py」を編集することでpathの設定を行う。pathの設定は path(アクセスするアドレス,呼び出す関数) のように行う。

#編集方法

from django.urls import path
from . import views

urlpatterns = [
    #localhost:8000/app_name/にアクセスすると、「app_name」ディレクトリ直下の「view.py」内の「index」という関数を実行できるようにする
    path('', views.index, name=index)

    #localhost:8000/app_name/homeにアクセスすると、「app_name」ディレクトリ直下の「view.py」内の「home」という関数を実行できるようにする
    path('/home/', views.home, name=home)

    #localhost:8000/app_name/1/addressにアクセスすると、
    #「app_name」ディレクトリ直下の「view.py」内の「address」という関数(引数にint型のidとString型のaddressを渡す)を実行できるようにする
    path('<int:id>/<address>', views.address, name=address)
]
