#メソッド集

メソッド 説明 戻り値の型
text.startswith('aaa') 文字列の前方一致 boolean
text.endswith('ccc') 文字列の後方一致 boolean
text.replace('old','new') 文字列のoldをnewに置換する void
text.replace('old','new',n) n回だけ文字列のoldをnewに置換する void
text.strip() 文字列の前後の空白を除去する void
text.rstrip() 文字列の前の空白を除去する void
text.lstrip() 文字列の後の空白を除去する void
text.upper() 文字列を大文字に変換した文字列を返す str
text.lower() 文字列を小文字に変換した文字列を返す str
text.swapcase() 大文字と小文字を反転させる str
text.capitalize() 先頭文字のみ大文字、その他は小文字に変換させる str
text.title() 単語の先頭を大文字に、その他は小文字に変換する str
text.isalnum() すべての文字が英数字かつ1文字以上あるかどうか boolean
text.isalpha() すべての文字が英字かつ1文字以上あるかどうか boolean
text.isascii() すべての文字がASCIIか空文字であるかどうか boolean
text.isdecimal() すべての文字が10進数の数字かつ1文字以上あるかどうか boolean
text.islower() すべての文字が小文字かつ1文字以上あるかどうか boolean
text.isupper() すべての文字が大文字かつ1文字以上あるかどうか boolean
text.isspace() すべての文字が空白文字かつ1文字以上あるかどうか boolean
text.split("区切り文字") 文字列を区切り文字で分割してリストで返す list
text.zfill(m) 文字列の長さがmになるように0で左詰めした文字列を返す str
text.rjust(n,'*') 文字列の長さをnとし、左側の空白を「*」で埋める str
text.ljust(n,'*') 文字列の長さをnとし、右側の空白を「*」で埋める str
text.center(n,'*') 文字列の長さをnとし、両側の空白を「*」で埋める str
str1.translate(str.maketrans({'old':'new'})) oldをnewに変換する(辞書式で複数一括変換可能) str
str1.find(str2) 部分文字列が登場する文字の位置を取得する (前方から) int
str1.rfind(str2) 部分文字列が登場する文字の位置を取得する (後方から) int
str1.count(str2) 部分文字列の登場回数を取得する int

例外処理

#例外クラス

例外クラス 意味
AttributeError 存在しない属性を指定した場合
IndexError 存在しない範囲のインデックスを指定した場合
KeyError 存在しないキーを指定した場合
TypeError 不正な型を指定した場合
ValueError 不正な値を指定した場合
ZeroDivisionError ゼロ除算した場合
BaseException すべての例外のスーパークラス
Exception システム終了以外のすべての組み込み例外のスーパークラス

#try-except構文 (例外変数を省略可能)

try:
    #例外が起こる可能性がある処理
except 例外クラス1 as 例外変数1:
    #処理

#try-except-else-finally構文

try:
    #例外が起こる可能性がある処理
except 例外クラス1 as 例外変数1:
    #処理
except 例外クラス2 as 例外変数2:
    #処理
else:
    #例外が発生しなかったときの処理
finally:
    #例外の有無に関わらず実行する終了時の処理

#例外を投げる

raise 例外オブジェクト

パッケージ

パッケージとは複数のモジュールをまとめたもの。実質的にはディレクトリ。

  • pkg
    • __init__.py (パッケージが呼び出されたときの処理を記述しておくためのファイル)
    • __main__.py (パッケージをpythonコマンドで実行したときの処理を記述しておくためのファイル)
    • app1.py (=モジュール)
    • app2.py (=モジュール)
    • subpkg (=サブパッケージ)
      • __init__.py
      • subapp1.py
      • subapp2.py
