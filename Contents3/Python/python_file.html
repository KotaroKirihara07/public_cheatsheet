ファイル

2024/02/10

このページではファイルの取り扱い方法についての説明を行う。

基本

#ファイルを開く

f = open("example.txt",読み書きモード)

#読み書きモード

読み書きモード 説明
"r" 読み込みモードで開く
"a" 追記モードで開く
"w" 書き込みモード開く(ファイルが存在しなかった場合は新しく作成)

#ファイル読み込みのメソッド集

メソッド 説明 戻り値
f.read() テキスト内のすべてのデータをひとつの変数に格納する str
f.readline() 1行ごとのデータ str
f.readlines() データを行単位で分割したリストとして格納する list

#ファイル書き込みのメソッド集

メソッド 説明 戻り値
f.write("wwww") 指定した文字列を書き込み、書き込まれた文字数を返す int
f.writelines(list1) 指定した文字列リストを1つずつ書き込む なし

#ファイルを閉じる

f.close()
