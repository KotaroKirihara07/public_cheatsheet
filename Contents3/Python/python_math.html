#数学定数

定数 説明
math.pi 3.14.....
math.e ネイピア数

#関数集

関数 説明 戻り値の型
math.exp(x) eのx乗 float
math.radians(180) 度数法から弧度法への変換 float
math.degrees(math.pi) 弧度法から度数法への変換 float
math.sin(x) サイン float
math.cos(x) コサイン float
math.tan(x) タンジェント float
math.asin(x) アークサイン float
math.acos(x) アークコサイン float
math.atan(x) アークタンジェント float
math.sqrt(3) 平方根 float
math.pow(x, y) xのy乗 float
math.log(x**3, x) log(真数、底) float
math.gcd(x, y) 最大公約数 int
math.lcm(x, y) 最小公倍数 int
math.fabs(x) 絶対値 int
math.ceil(x) 切り上げ int
math.floor(x) 切り下げ int
math.factorial(x) 階乗 int
math.isclose(x, y) xとyの差が1e-9以内かどうか boolean
