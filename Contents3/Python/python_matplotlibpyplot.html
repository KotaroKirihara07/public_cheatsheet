matplotlib.pyplot

2024/02/12

このページではmatplotlib.pyplotについての説明を行う。

モジュールの読み込み

#モジュールの読み込み

import matplotlib.pyplot as plt

基本 (オブジェクト指向スタイル)

#figureの生成

fig = plt.figure()

#figureをN行M列に分割してn番目にaxesを追加する

ax = fig.add_subplot(N,M,n)

#表示

ax.plot(x,y)
plt.show()

散布図 (オブジェクト指向スタイル)

#散布図を作成

ax.scatter(x,y,オプションパラメータ)
plt.show()

#オプションパラメータ集

パラメータ 説明 設定値
s マーカーサイズ 数値
c マーカーの色 '#aaaaFF'
marker マーカーの形 '.' (点)、 '*' (星印)、 '+' (+)、 'D' (ひし形)
alpha マーカーの透明度 0~1
linewidths マーカーのエッジ線の太さ 数値
edgecolors マーカーのエッジ線の色 'black' 、 'blue' 、 'red' 、 'green' 、 'white'

棒グラフ (オブジェクト指向スタイル)

#棒グラフを作成

ax.bar(x,y,オプションパラメータ)
plt.show()

#オプションパラメータ集

パラメータ 説明 設定値
width 棒の幅 数値
bottom 棒の下側の位置を指定 数値
color 棒の色を指定 16進数カラーコード
edgecolor 棒の枠線の色を指定 16進数カラーコード
linewidths マーカーのエッジ線の太さ 数値
tick_label 横軸のラベル str
align 棒の位置を指定 center or edge

折れ線グラフ (オブジェクト指向スタイル)

#折れ線グラフを作成

ax.plot(x,y,オプションパラメータ)
plt.show()

#オプションパラメータ集

パラメータ 説明 設定値
fmt 線の種類 "-" (実線)、 "--" (破線)、 "-." (点+破線)、 ":" (点線)
c 線の色 16進数カラーコード
linewidth 線の太さ 数値
alpha 透明度 0~1
mraker マーカーの形 '.' (点)、 '*' (星印)、 '+' (+)、 'D' (ひし形)

円グラフ (オブジェクト指向スタイル)

#円グラフを作成

ax.pie(x,オプションパラメータ)
plt.show()

#オプションパラメータ集

パラメータ 説明 設定値
labels 各要素のラベルの配列 リスト
colors 各要素の色の配列 リスト
counterclock 反時計回り(True) or 時計回り(False) True or False
startangle 開始角度 数値

ヒストグラム (オブジェクト指向スタイル)

#ヒストグラムを作成

ax.hist(x,オプションパラメータ)
plt.show()

#オプションパラメータ集

パラメータ 説明 設定値
bins 階級数 or 階級リスト or 階級の分割方式 数値
density Trueの場合面積が1になる True or False
color 色を指定 16進数カラーコード
ec 境界色を指定 16進数カラーコード
alpha 透明度を指定 0~1

グラフの汎用要素の設定 (オブジェクト指向スタイル)

#メソッド集

メソッド 説明
ax.set_title("タイトル") グラフタイトルを設定する
ax.grid(True) グリッドオン
ax.set_xlim(0,100) x軸の範囲を指定
ax.set_ylim(0,100) y軸の範囲を指定
ax.set_xlabel("ラベル名") x軸のラベルを設定
ax.set_ylabel("ラベル名") y軸のラベルを設定
ax.legend(["f(x)=x","g(x)=x"]) グラフの凡例を設定
ax.set_xticks([0,1,2,3,4,5]) x軸の目盛を指定する
ax.set_yticks([0,1,2,3,4,5]) y軸の目盛を指定する

基本 (MATLABスタイル)

#表示

plt.plot(x,y)
plt.show()

#サイズの変更

plt.figure(figsize=(4,4))

#フォントの変更

plt.rcParams["font.size"] = 18

#はみださないようにする

plt.tight_layout()

グラフの汎用要素の設定

#メソッド集

メソッド 説明
plt.title("タイトル") グラフタイトルを設定する
plt.xlim(0,100) x軸の範囲を指定
plt.ylim(0,100) y軸の範囲を指定
plt.xlabel("ラベル名") x軸のラベルを設定
plt.xlabel(r'$x_{1} + y_{1}$') Latex記法でx軸ラベルを作成する
plt.ylabel("ラベル名") y軸のラベルを設定
plt.ylabel(r'$f(x) = 2x$') Latex記法でy軸ラベルを作成する
plt.legend(["f(x)=x","g(x)=x"]) グラフの凡例を設定
plt.xticks([0,1,2,3,4,5]) 表示するx軸の目盛を指定する
plt.xticks(元のリスト,新しいリスト) 表示するx軸の目盛を変更する
plt.yticks([0,1,2,3,4,5]) 表示するy軸の目盛を指定する
plt.yticks(元のリスト,新しいリスト) 表示するy軸の目盛を変更する
plt.xscale("log") x軸をlogスケールに変換
plt.yscale("log") y軸をlogスケールに変換
plt.savefig("img.pdf") 図の保存を保存する

その他

#x軸縦書き(90度回転)

plt.xticks(rotation=90)

#フォントサイズの変更

plt.xlabel(x,fontname="MS Gothic")
plt.ylabel(x,fontname="MS Gothic")
plt.xlabel(x,fontname="Segoe UI")
plt.ylabel(x,fontname="Meirio")

#軸の数字をオフセットを使わずに表現する

import matplotlib.ticker as ticker
plt.gca().get_xaxis().get_major_formatter().set_useOffset(False)
plt.gca().get_yaxis().get_major_formatter().set_useOffset(False)

#軸の数字が必ず整数になるようにする

import matplotlib.ticker as ticker
plt.gca().get_xaxis().set_major_locator(ticker.MaxNLocator(integer=True))
plt.gca().get_yaxis().set_major_locator(ticker.MaxNLocator(integer=True))
