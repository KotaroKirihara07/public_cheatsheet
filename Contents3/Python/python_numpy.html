ndarray

ndarrayはNumpyの配列であり、リストと比較して処理速度が速く、数学的な演算等の便利な機能をもっている。

#ndarrayの生成

メソッド 説明 戻り値の型
np.array(1次元リスト) 1行n列の行列の生成 ndarray
np.array(2次元リスト) 行列の生成 ndarray
np.arange(start,stop,step) startから(stop-1)までのstep区切りの1行n列の行列の生成 ndarray
np.linspace(start,stop,n) startからstopまで等間隔の要素数nの1次元配列 ndarray
np.date_range("YYYY/MM/DD","yyyy/mm/dd",freq="30s") YYYY/MM/DDからyyyy/mm/ddまで30秒間隔の1次元配列 ndarray
np.full(shape=(N,M),fill_value=3) N行M列の要素のすべてが3の行列 ndarray
np.eye(N) N行N列の単位行列 ndarray
np.zeros((N,M)) N行M列のゼロ行列 ndarray
np.tri(N) N行N列の三角行列 ndarray
np.ones((N,M)) N行M列の要素がすべて1の行列 ndarray

乱数

#シードを固定して、ジェネレータの生成する

rng = np.random.default_rng(12345)

#乱数の生成

メソッド 説明 戻り値の型
rng.random(n) 0以上1未満の一様分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.randint(0,10,n) 0以上10以下の離散一様分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.uniform(0,10,n) 0以上10より小さいの連続一様分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.randn(n) 標準正規分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.normal(loc,scale,n) 平均loc、標準偏差scaleの正規分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.logistic(loc,scale,n) 平均loc、標準偏差scaleのロジスティック分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.binomial(N,p,n) 試行回数N、確率pの二項分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.exponential(a,n) パラメータがaの指数分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.beta(a,b,n) パラメータa,bのベータ分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.gamma(a,b,n) 形状母数がa、尺度母数がbのガンマ分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.chisquare(a,n) 自由度aのχ二乗分布に従う乱数をn要素もつndarrayの生成 ndarray
rng.choice(a=value_list,size=n,p=prob_list) value_list[i]である確率がprob_list[i]の確率分布に従う乱数をn要素もつndarrayの生成 ndarray
※上記の変数の個数nを(N,M)とすることでN行M列のランダム行列に変更できる
