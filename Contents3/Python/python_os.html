このページではosモジュールについての説明を行う。

モジュールの読み込み

#モジュールの読み込み

import os

ディレクトリ操作

#メソッド集

メソッド 説明
os.getcwd() 現在のディレクトリを取得
os.path.abspath(path) 絶対パスを取得
os.path.relpath(path,起点となるpath) 相対パスを取得
os.path.exists(path) パスの存在を確認(True or False)
os.mkdir("./new_dir) 新規ディレクトリの作成
os.chdir("C:/data") ディレクトリの移動
os.chdir("/..") 一つ上の階層に移動
os.rmdir("./test1") ディレクトリの削除
os.path.isdir(path) 指定したパスがディレクトリかどうか(True or False)

ファイル操作

#メソッド集 (os)

メソッド 説明
os.remove("./myfile.txt") ファイルの削除
os.listdir(path) ディレクトリ直下のファイル名のリストを取得する
file_name = [i for os.listdir(path) if 条件] 特定の条件に合致したファイルのリストを作成する

#メソッド集 (os.path)

メソッド 説明
os.path.exists("./myfile.txt") ファイルが存在するかどうか調べる(True or False)
os.path.isfile(path) 指定したパスがファイルかどうか(True or False)
file_name, file_ext = os.path.splitext(path) 拡張子を取得する
os.path.abspath("./myfile.txt") 絶対パスを取得する
os.path.getatime("./myfile.txt") 最終アクセス時刻を取得する
os.path.getmtime("./myfile.txt") 最終更新時刻を取得する
os.path.getsize("./myfile.txt") サイズ(バイト数)を返す
