このページではpandasの説明を行う。

Series : pandasで扱うことのできるデータ形式の一種であり、1次元配列で単一の列からなる表のようなイメージ。

DataFrame: 行と列からなる表形式の配列データであり、Seriesを複数集めたものというイメージ。

モジュールの読み込み

#モジュールの読み込み

import pandas as pd

データの生成

#Seriesの生成

S = pd.Series(1次元データ配列, index=index_list)

#辞書からDataFrameを生成する (indexは省略可)

data = {"column1":1次元データ配列1, "column2":1次元データ配列2}
df = pd.DataFrame(data, index=index_list)

#2次元データ配列からDataFrameを生成する (indexは省略可)

df = pd.DataFrame(2次元データ配列, columns=columns_list, index=index_list)

ファイルからの読み込み・ファイルへの書き込み

#xxxファイルの読み込み、xxxファイルへの書き込み (xxx = [csv(tsv), pickle, excel, json, parquet])

df = pd.read_xxx("FILE_PATH")
df.to_xxx("FILE_PATH")

#読み込み用オプションパラメータ

パラメータ 説明
sep 区切り文字、デフォルトはカンマ (sep="\t"とするとtsvに対応)
header ヘッダ行番号(デフォルトは0)、ヘッダなしの場合は None を指定
dtype 列ごとの型を辞書で指定

#書き込み用オプションパラメータ

パラメータ 説明
sep 区切り文字、デフォルトはカンマ (sep="\t"とするとtsvに対応)
index indexの出力要否をbool型で指定
index_label indexを出力する場合のカラム名

#メソッド集

メソッド 説明 戻り値の型
df.loc[] Dataframe
df.iloc[] Dataframe
df.count() 各列のデータ件数 int
df.mean() 各列のデータの平均 var

df["col1"] or df.col1

df2 = df.groupby(["col1"])

#特定の列に対するメソッド集

メソッド 説明 戻り値の型
df["col1"].values ある列をndarray型に変換して返す ndarray
df["col1"].unique() ある列のユニークな値のリストを取得する list
df["col1"].value_counts() ある列のユニークな値とその値が表れた回数を取得する ???
df["col1"].nunique() ある列のユニークな値の数を取得する int

DataFrameの加工

#メソッド集

メソッド 説明
df.drop(idx) 特定の行の削除
df.drop(["col1", "col2"], axis=1) 特定の列の削除
df.drop_duplicates() 重複した行の削除
df.replace(old_value, new_value) 値の置換
df.sort_values(by=["col1", "col2"], ascending=[True, False]) DataFrameのソート (col1を基準に昇順、col2を基準に降順)
df.reset_index(drop=True) インデックスをリセットする
df.dropna() 欠損値のある行の除去 (subset=["col1", "col2"]として特定列のみに限定可能)
df.rename(columns={"old_col1":"new_col1", "old_col2":"new_col2"}) 列名を変更する

#関数集

関数 説明 戻り値の型
pd.isnull(df) 欠損値があるか調べる bool
pd.concat([df1, df2]) DataFrameの縦方向の結合 (axis=1とすると横結合) DataFrame
pd.merge(df1, df2, on="col1") col1で内部結合 (how="outer"とすると外部結合) void

DataFrameのフィルタリング

#フィルタリング (条件1つ)

condition = (条件)
df[condition]

#bool型シーケンスによるフィルタリング (1行目と3行目のみ抽出)

condition = [True, False, True, False]
df[condition]

#正規表現によるフィルタリング

condition = df.列名.str.contains('正規表現')
df[condition]

#andフィルタリング(条件2つ)

condition1 = (条件1)
condition2 = (条件2)
df[condition1 & condition2]

#orフィルタリング(条件2つ)

condition1 = (条件1)
condition2 = (条件2)
df[condition1 | condition2]

#indexを削除して検索結果のみString型で取得する

s = df[condition1 | condition2].to_string(index=False)

#クエリでフィルタリング

df.query("col1 > 10")
