このページではpandasの説明を行う。

Series : pandasで扱うことのできるデータ形式の一種であり、1次元配列で単一の列からなる表のようなイメージ。

DataFrame: 行と列からなる表形式の配列データであり、Seriesを複数集めたものというイメージ。

モジュールの読み込み

#モジュールの読み込み

import pandas as pd

データの生成

#Seriesの生成

S = pd.Series(1次元データ配列,index=index_list)

#辞書からDataFrameを生成する (indexは省略可)

data = {"column1":1次元データ配列1,"column2":1次元データ配列2}
df = pd.DataFrame(data,index=index_list)

#n次元データ配列からDataFrameを生成する (indexは省略可)

df = pd.DataFrame(n次元データ配列,columns=columns_list,index=index_list)

ファイルからの読み込み・ファイルへの書き込み

#csvファイルの読み込み、csvファイルへの書き込み

df = pd.read_csv("file_name.csv")
df.to_csv("file_name.csv",index=False)

#tsvファイルの読み込み、tsvファイルへの書き込み

df = pd.read_csv("file_name.tsv",sep='\t')
df.to_csv("file_name.tsv",sep='\t',index=False)

#pickleファイルの読み込み、pickleファイルへの書き込み

df = pd.read_pickle("file_name.pkl")
df.to_pickle("file_name.pkl")

#読み込み用オプションパラメータ

パラメータ 説明
sep 区切り文字、デフォルトはカンマ
header ヘッダ行番号(デフォルトは0)、ヘッダなしの場合は None を指定
dtype 列ごとの型を辞書で指定

#書き込み用オプションパラメータ

パラメータ 説明
sep 区切り文字、デフォルトはカンマ
index indexの出力要否をbool型で指定
index_label indexを出力する場合のカラム名

DataFrameの基本統計情報

#メソッド集

メソッド 説明
df.count() 各列のデータ件数
df.mean() 各列のデータの平均
df.std() 各列の標準偏差
df.max() 各列の最大値
df.min() 各列の最小値
df.var() 各列の分散
df.sample() 各列からのランダムサンプリング
df.describe() 各列の基本統計情報を一括取得

DataFrameのデータの取得

#列データの取得

df["column名"] or df.column名

#行データの取得

df.loc[index名] or df.iloc[index番号]

#行・列を指定したデータの取得

df.at[index名,column名] or df.iat[index番号,column番号] or df.column名.index名

DataFrameの加工

#特定の行の削除

df.drop(2)

#値の置換

df.replace(置換前,置換後)

#欠損値を0に置換する

df.replace(np.Nan,0)

#DataFrameのソート (col1を基準に昇順)

df.sort_values(["col1","col2","col3"],ascending=[True,False,False])

#DataFrameの縦方向の結合 (indexをリセット)

df5 = pd.concat([df3,df4]).reset_index(drop=True)

#DataFrameの横方向の結合

df5 = pd.concat([df3,df4],axis = 1)

#欠損値があるか調べる (True or False)

pd.isnull(df).any()

#欠損値のある行の除去

df.dropna()

#特定の列に欠損値がある行の削除

df.dropna(subset=["列名1","列名2"])

DataFrameのフィルタリング

#フィルタリング (条件1つ)

condition = (条件)
df[condition]

#bool型シーケンスによるフィルタリング (1行目と3行目のみ抽出)

condition = [True,False,True,False]
df[condition]

#正規表現によるフィルタリング (1行目と3行目のみ抽出)

condition = df.列名.str.contains('正規表現')
df[condition]

#andフィルタリング(条件2つ)

condition1 = (条件1)
condition2 = (条件2)
df[condition1 & condition2]

#orフィルタリング(条件2つ)

condition1 = (条件1)
condition2 = (条件2)
df[condition1 | condition2]

#indexを削除して検索結果のみString型で取得する

s = df[condition1 | condition2].to_string(index=False)

DataFrameの集計

#メソッド集

メソッド 説明 戻り値の型
df.groupby([column名]).min() 指定したcolumn名でグループ化し、他のそれぞれの列の最小値を出力 DataFrame
df.groupby([column名]).max() 指定したcolumn名でグループ化し、他のそれぞれの列の最大値を出力 DataFrame
df.groupby([column名]).sum() 指定したcolumn名でグループ化し、他のそれぞれの列の合計値を出力 DataFrame
df.groupby([column名]).mean() 指定したcolumn名でグループ化し、他のそれぞれの列の平均値を出力 DataFrame
df.groupby([column名]).var() 指定したcolumn名でグループ化し、他のそれぞれの列の分散を出力 DataFrame
df.groupby([column名]).std() 指定したcolumn名でグループ化し、他のそれぞれの列の標準偏差を出力 DataFrame
