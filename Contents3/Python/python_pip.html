pip

2024/02/10

このページでは外部ライブラリのインストールやアンインストール方法についての説明を行う。

pip

#パッケージをインストールする

pip install numpy

#パッケージをアンインストールする

pip uninstall -y numpy

#インストールされているパッケージを確認する

pip freeze

#インストールされているパッケージの保存場所を出力する

pip show パッケージ名

#パッケージのアップグレード

pip install --upgrade numpy
