psycopg2

2024/12/08

psycopg2はPostgreSQLへの接続モジュール。

モジュールの読み込み

#モジュールの読み込み

import psycopg2

接続

#接続情報

con_info = {"user":"db user", "password":"db password", "host":"localhost", "dbname":"sample"}

接続パラメータ 意味
user ユーザ
password パスワード
host ホスト
dbname データベース
port ポート

コネクション操作

メソッド 処理
psycopg2.connect(**con_info) 接続
conn.commit() コミット
conn.close() クローズ
conn.rollback() ロールバック
conn.cursor() カーソルの取得

カーソル操作

メソッド 処理
cur.execute(SQL文,オプション) SQLの実行
cur.close() クローズ
cur.fetchone() 1行フェッチ
cur.fetchall() 全件フェッチ

テンプレート

import psycopg2

#接続情報
con_info = {"user":"db user", "password":"db password", "host":"localhost", "dbname":"sample"}

#接続
with psycopg2.connect(**con_info) as con:

     #カーソルの取得
     with con.cursor() as cur:

         sql = "SQL文"
         cur.execute(sql)
         #結果の取得
         rows = cur.fetchall()
