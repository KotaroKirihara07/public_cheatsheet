pulp

2023/08/02

線形計画問題を解くためのパッケージです。

モジュールの読み込み

#モジュールの読み込み

from pulp import *

数理モデルの作成

#最小化問題

m = LpProblem()

#最大化問題

m = LpProblem(sense=LpMaximize)

変数の作成

#連続変数

x1 = LpVariable('x1', lowBound=0)

#0-1変数

x2 = LpVariable('x2', cat=LpBinary)

目的関数と制約条件

#目的関数

m += 式

#制約条件

m += 式 == 数値
m += 式 <= 数値
m += 式 >= 数値

実行と結果の出力

#ソルバーの実行

m.solve()

#最適解

value(x1)
value(x2)

#最適値

value(m.objective)
