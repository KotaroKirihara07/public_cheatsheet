import random

乱数の生成

#メソッド集

メソッド 説明 戻り値の型
random.random() 0以上1未満の乱数を生成 float
random.uniform(a,b) a以上b以下の乱数を生成 float
random.randint(a,b) a以上b以下の乱数を生成 int
random.randrange(a,b,n) a以上b以下のうちnステップごとのリストから一様分布に従う乱数を生成 int
random.betavariate(alpha,beta) ベータ分布に従う乱数を生成する float
random.expovariate(lambda) 指数分布(lambdaは平均の逆数)に従う乱数を生成する float
random.gammavariate(alpha,beta) ガンマ分布に従う乱数を生成 float
random.gauss(mu,sigma) ガウス分布(muが平均、sigmaが標準偏差)に従う乱数を生成 float
random.lognormvariate(mu,sigma) 対数正規分布に従う乱数を生成 float
random.normalvariate(mu,sigma) 正規分布に従う乱数を生成 float
random.paretovariate(alpha) パレート分布(形状パラメータがalpha)に従う乱数を生成 float
random.weibullvariate(alpha,beta) ワイブル分布(尺度パラメータがalpha、形状パラメータがbeta)に従う乱数を生成 float
random.triangular(low,high,mode) 最頻値modeをもつlow以上high以下の乱数を生成 float

説明

メソッド 説明 戻り値の型
random.choice(list1) リストのうちランダムな要素1つを返す var
random.sample(list1,len(list1)) リストをシャッフルして新しいリストを返す list
random.shuffle(list1) リストをシャッフルする(戻り値なし) void
