インストール

Python (for Windows)

  1. 公式サイト( https://www.python.org/ )からダウンロード
  2. パスを自動で通すようにしてインストール
  3. python --versionと入力してインストールに成功したか確認する

Python (for Red Hat Linux)

  1. sudo yum install python39
  2. python --versionと入力してインストールに成功したか確認する

Python (for Debian Linux)

  1. sudo apt-get install python3.9
  2. python --versionと入力してインストールに成功したか確認する

jupyter lab (推奨)

  1. python -m pip install --upgrade pip
  2. pip install jupyterlab
  3. jupyter labで起動

jupyter notebookのインストール (非推奨)

  1. python -m pip install --upgrade pip
  2. python -m pip install jupyter
  3. jupyter notebookで起動

ファイルの実行

@VSCode

  1. VSCodeでPythonの拡張機能をインストールする
  2. ディレクトリを作成する
  3. ディレクトリに移動し、ファイルを作成する
  4. [表示]、[ターミナル] の順に選択して、Visual Studio Code 内の組み込みターミナルを開く
  5. python file_name.py

@コマンドプロンプト/ターミナル

  1. ファイルがあるディレクトリへ移動
  2. python filename.py

インタプリタの切り替え

pyenv

#Python (3.11.9)をインストールする

pyenv install 3.11.9

#インストール可能なPythonのバージョンを確認する

pyenv install --list

#pyenvを使用してインストールされたPythonのバージョンを確認する

pyenv versions

#pyenvでインストールしたPythonを削除する

pyenv uninstall 3.11.9

パッケージの切り替え

pipによるパッケージの導入状態をプロジェクトごとに独立させることができる。

venv

#仮想環境のセットアップ (for Linux)

python -m venv /path/to/new/virtual/environment
source /path/to/new/virtual/environment/bin/activate

#仮想環境のセットアップ (for Windows)

py -m venv /path/to/new/virtual/environment
.\path\to\new\virtual\environment\Scripts\activate

#deactivate

deactivate
