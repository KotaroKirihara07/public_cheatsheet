Python

インストール

  1. 公式サイト( https://www.python.org/ )からダウンロード
  2. パスを自動で通すようにしてインストール
  3. python --versionと入力してインストールに成功したか確認する

実行

  1. ファイルがあるディレクトリへ移動
  2. python filename.py

jupyter labのインストール (推奨)

  1. python -m pip install --upgrade pip
  2. pip install jupyterlab
  3. jupyter labで起動

jupyter notebookのインストール (非推奨)

  1. python -m pip install --upgrade pip
  2. python -m pip install jupyter
  3. jupyter notebookで起動

VSCodeでPythonファイルの実行

  1. VSCodeでPythonの拡張機能をインストールする
  2. ディレクトリを作成する
  3. ディレクトリに移動し、ファイルを作成する
  4. [表示]、[ターミナル] の順に選択して、Visual Studio Code 内の組み込みターミナルを開く
  5. python file_name.py
