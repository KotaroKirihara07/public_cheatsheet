Systemクラス 、 Stringクラス 、 Mathクラス

インスタンス化の必要がない。

Systemクラス

出力を扱うためのクラス。

#出力 (改行あり)

System.out.println(12345);

#出力 (改行なし)

System.out.print(12345);

Stringクラス

文字列を扱うためのクラス。

#文字列の生成

String str = "abcde";
var str1 = new String("abcde");
var str2 = "abcde";

#文字列への変換

String str1 = String.valueOf(123);

#メソッド集

メソッド 説明 戻り値の型
str1.codePointCount(0,str1.length()) 文字列の長さを調べる int
str1.isEmpty() 文字列が空であるかどうか boolean
str1.isBlank() 空文字列または空白のみかどうか boolean
str1.indexOf(str2) 特定の文字列が最初に登場する文字列のインデックスを返す (存在しなければ-1を返す) int
str1.indexOf(str2,n) 特定の文字列が最初に登場する文字列のインデックスを返す (検索開始位置をnとする, 存在しなければ-1を返す) int
str1.lastIndexOf(str2) 特定の文字列が最後に登場する文字列のインデックスを返す (存在しなければ-1を返す) int
str1.contains(str2) 文字列を含むかどうか boolean
str1.startsWith(str2) str1先頭の文字列がstr2と一致するかどうか boolean
str1.endsWith(str2) str1末尾の文字列がstr2一致するかどうか boolean
str1.compareTo(str2) 文字列が一致しているかどうか (0で一致) int
str1.compareToIgnoreCase(str2) 大文字小文字区別しないで文字列が一致しているかどうか (0で一致) int
str1.equals(str2) 文字列が一致しているかどうか (true or false) boolean
str1.equalsIgnoreCase(str2) 大文字小文字区別しないで文字列が一致しているかどうか boolean
str1.strip() 文字列の前後の空白を除去する void
str1.stripLeading() 文字列の前の空白を除去する void
str1.stripTrailing() 文字列の後の空白を除去する void
str1.substring(2,5) 部分文字列を取得する String
str1.split(",") 特定の区切り文字で分割する (カンマ区切り) String
String.join(",",str1) 特定の区切り文字で連結する (カンマで連結) String
str1.replaceAll(String a, String b) str1の"a"をすべて"b"に変換する。(第一引数を正規表現にすることも可能) void

#Formatメソッド

String.format("%sは%d歳です。","私","20")

#変換文字

変換文字 説明
%b,%B ブール値
%h,%H ハッシュコードを16進数に変換
%s,%S 文字列
%c,%C Unicode文字
%d 10進数
%o 8進数
%x,%X 16進数
%e,%E 浮動小数点数形式の10進数
%f 10進小数点数
%t,%T 日付、時刻変換文字の接頭辞
%% '%'
%n プラットフォーム固有の行区切り文字

Mathクラス

数学的演算を行うためのクラス。

#メソッド集

メソッド 説明 戻り値の型
Math.abs(x) 絶対値 var
Math.min(a,b) 最小値 var
Math.max(a,b) 最大値 var
Math.ceil(x) 数値の切り上げ double
Math.floor(x) 数値の切り捨て double
Math.round(x) 数値を四捨五入 int
Math.sqrt(x) 平方根 double
Math.cbrt(x) 立方根 double
Math.pow(x,y) xのy乗 double
Math.sin(x) サイン double
Math.sin(x) サイン double
Math.cos(x) コサイン double
Math.tan(x) タンジェント double
Math.asin(x) アークサイン double
Math.acos(x) アークコサイン double
Math.atan(x) アークタンジェント double
Math.toRadians(x) 角度をラジアンに変換 double
Math.toDegrees(x) ラジアンを度に変換 double
Math.exp(x) eのx乗 double
Math.log(x) 自然対数 double
Math.log10(x) 底が10の対数 double
