LocalDateTimeクラス

日付や時刻を操作するためのクラス。

#import宣言

import java.time.LocalDateTime;

#現在の時刻を生成する

var T = LocalDateTime.now();

#任意の時刻 (2020年1月1日13時20分30.123456789秒)を生成する

var T = LocalDateTime.of(2020,1,1,13,20,30,123456789);

#メソッド集

メソッド 説明 戻り値の型
t1.equals(t2) t1とt2が等しいかどうか boolean
t1.isBefore(t2) t1 < t2かどうか boolean
t1.isAfter(t2) t1 > t2かどうか boolean
T.getYear() 年を取得 int
T.getMonthValue() 月を取得 int
T.getDayOfMonth() 日を取得 int
T.getDayOfWeek() 曜日を取得 var
T.getDayOfYear() その年の何日目か取得 int
T.getHour() 時を取得 int
T.getMinute() 分を取得 int
T.getSecond() 秒を取得 int
T.getNano() ナノ秒を取得 int

Periodクラス

日付間隔を扱うためのクラス。

#import宣言

import java.time.Period;

#Periodオブジェクトを生成

var period = Period.between(t1.toLocalDate(),t2.toLocalDate());

#メソッド集

メソッド 説明 戻り値の型
period.getYears() 年数 int
period.getMonths() 月数 int
period.getDays() 日数 int

Durationクラス

時間間隔を扱うためのクラス。

#import宣言

import java.time.Duration;

#Durationメソッドの生成

var duration = Duration.between(t1,t2);

#メソッド集

メソッド 説明 戻り値の型
duration.toDays() 日数 long
duration.toHours() 時間数 long
duration.toMinutes() 分数 long
duration.toSeconds() 秒数 long
duration.toMillis() ミリ秒数 long
duration.toNanos() ナノ秒数 long
