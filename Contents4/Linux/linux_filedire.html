awk

空白やカンマなどで区切られたテキストを列として扱い、出力したり特定の処理をしたりするためのコマンド。

awk [オプション] 'パターン {アクション}' <ファイル名>

#コロンで区切られたファイル内の2列目と3列目を表示

awk -F':'{print $2,$3}' text.txt

bzip2

ファイルの圧縮, 展開するためのコマンド。

bzip2 [オプション] <ファイル/ディレクトリ>

#圧縮する

bzip2 file1

#展開する

bzip2 -d file1.bz2

cat

ファイルの内容を表示するコマンド。

cat [オプション] <ファイル名>

#ファイルの内容を表示

cat text.txt

#行数付きでファイルの内容を表示

cat -n text.txt

#表示した内容を他のファイルの末尾に書き込む

cat text1.txt >> text2.txt

cd

ディレクトリを変更するためのコマンド。

cd <ディレクトリ名>

cp

ファイル,ディレクトリをコピーするためのコマンド。

cd [オプション] <コピー元> <コピー先>

#ファイルのコピー

cp file1 file2

#複数のファイルをディレクトリにコピーする

cp file1 file2 file3 dir1

#特定の拡張子をもつファイルをディレクトリにコピー

cp *.txt sample

#確認した後ファイルのコピー

cp -i file1 file2

#ディレクトリのコピー

cp -r dir1 dir2

cut

入力行の一部を切り出して出力するコマンド。

cut -d <区切り文字> -f <フィールド番号> [ファイル名]

#「:」で区切り, 7番目のフィールドを切り出して表示

cut -d : -f 7 file1

diff

ファイルの差分を表示するためのコマンド。

diff [オプション] <比較元ファイル> <比較先ファイル>

find

ファイル探索のためのコマンド。

find <検索開始のディレクトリ> <検索条件> <アクション>

#現在のディレクトリ内のファイルの探索してファイル名を表示

find . -name file.txt -print

#現在のディレクトリ内の特定の拡張しを持つファイルを探索してファイル名を表示

find . -name *.txt -print

gedit

ファイルを編集するためのコマンド。

gedit <ファイル名>

grep

ファイル内の文字列を検索するコマンド。

grep [オプション] <検索パターン> <ファイル名>

#「aaa」を含む行を表示

grep aaa file1

#行番号付きで「aaa」を含む行を表示

grep -n aaa file1

#大文字と小文字を区別しないで「aaa」を含む行を表示

grep -i aaa file1

#正規表現により「aaa」を検索

grep 'a*' file1

#正規表現により「aaa」を検索

grep 'a..' file1

#括弧内のいずれかと一致する特定の文字を指定して検索(c or d or e)

grep 'ab[cde]efg' file1

#括弧内を除外した特定の文字を指定して検索(c以外)

grep 'ab[^c]efg' file1

#行頭が特定の文字の行を表示

grep '^abc' file1

#行末が特定の文字の行を表示

grep 'abc$' file1

gzip

ファイルの圧縮, 展開をするためのコマンド。

gzip [オプション] <ファイル/ディレクトリ>

#圧縮する

gzip file1

#圧縮ファイルを展開する

gzip -d file.gz

head

ファイル内の先頭数行を表示するためのコマンド。

head [オプション] <ファイル名>

#先頭3行を表示

head -n 3 file1

less

ファイルの内容をスクロール表示するためのコマンド。

less [オプション] <ファイル名>

#スクロール表示する

less file1.txt

locate

ファイル名でファイルの検索するためのコマンド。

locate [オプション] <ファイル名>

#test.txtを検索(検索できなかったらupdatedbコマンドを実行したあと実行)

locate test.txt

ls

ファイル名やディレクトリ名を表示するためのコマンド。

ls [オプション] [ディレクトリ/ファイル]

オプション 説明
-a 隠しファイルを含むすべてのディレクトリ/ファイルを表示
-F ファイル種別の表示

mkdir

ディレクトリを作成するためのコマンド。

mkdir [オプション] <作成するディレクトリ>

オプション 説明
-p 途中のディレクトリとともに新しいディレクトリを作成

mv

ファイル、ディレクトリ移動のためのコマンド。

mv [オプション] <移動元> <移動先>

#ファイル名の変更

mv file1 file2

#ファイルをディレクトリに移動する

mv file1 dir1

#複数ファイルの移動

mv file1 file2 file3 dir2

#確認した後ファイル名の変更

mv -i file1 file2

#ディレクトリの移動

mv dir1 dir2

pwd

現在のディレクトリを表示するためのコマンド。

pwd

rm

ファイル,ディレクトリ削除のためのコマンド。

rm [オプション] <削除するファイル/ディレクトリ> <削除するファイル/ディレクトリ> ...

オプション 説明
-r ディレクトリを削除
-i 確認後にファイルを削除

rmdir

空のディレクトリを削除するためのコマンド。

rmdir <ディレクトリ名>

sed

文字列を置換するためのコマンド。

sed [オプション] <スクリプト> <対象ファイル>

#1行目を削除

sed 1d file1

#2行目から5行目を削除

sed 2,5d file1

#3行目から最終行を削除

sed '3,$d' file1

#全行を削除

sed d file1

#1行目だけを表示する

sed -n 1p file1

#「aaa」を「bbb」に置換する

sed 's/aaa/bbb/' file1

sort

ファイルの内容をソートするコマンド。

sort [オプション] <ファイル名>

オプション 説明
-n 数値順に並べる
-r 逆順に並べる

tail

末尾部分を表示するコマンド。

tail [オプション] <ファイル名>

#末尾10行を表示

tail file1

#末尾1行のみ表示

tail -n 1 file1

#ファイルを監視する

tail -f file1

tar

アーカイブ作成のためのコマンド。

tar [オプション] <アーカイブファイル> <アーカイブ元ファイルパス>

#dir1ディレクトリにアーカイブする

tar -cf dir1.tar dir1

#アーカイブファイルの内容を表示する

tar -tf dir1.tar

#アーカイブファイルの展開

tar -xf dire1.tar

touch

ファイル作成のためのコマンド。

touch <新しいファイル名1> <新しいファイル名2> ...

tr

文字を置き換えるコマンド。

tr <置換前の文字> <置換後の文字>

#abcをABCに置換する

tr abc ABC

#小文字を大文字へ置換

tr a-z A-Z

#文字の削除

tr -d <削除文字>

tree

ディレクトリの内容をツリー状に表示するためのコマンド。

tree [オプション] [ディレクトリ]

#ディレクトリの内容をツリー状に表示

tree

#ディレクトリのみツリー状に表示

tree -d

#指定した階層だけ表示

tree -L 6

#隠しファイルも含めすべてのディレクトリを表示

tree -a

unique

重複を除いた内容を返すコマンド。

unique [オプション] <ファイル名>

#重複を除いた内容

unique file1

#重複行を数える

unique -c file1

updatedb

locateコマンド用のファイル名データベースを生成するためのコマンド。

updatedb [オプション]

#locateコマンド用のファイル名データベースを生成する

updatedb

wc

入力ファイルの行数,単語数,バイト数を数えるコマンド。

wc [オプション] <ファイル名>

#行数を表示

wc -l file1

#単語数を表示

wc -w file1

#バイト数を表示

wc -c file1

zip

ファイルをアーカイブ, 圧縮するためのコマンド。

zip [オプション] <ファイル名>

#zipファイルの作成

zip -r file1.zip file1

#zipファイルの内容を確認

zipinfo file1.zip

#zipファイルの展開

unzip file1.zip

#パスワードつきzipファイルの作成

zip -er file1.zip file1
