curl

指定したURL宛てにWeb接続を行うためのコマンド。

curl [オプション] URL

#www.example.comの内容を取得する (httpのGETメソッドに相当)

curl https://www.example.com

#AWS CloudShellのグローバルIPアドレスを取得する

curl http://checkip.amazonaws.com/

#JSON形式のリクエストボディを使用したPOSTリクエスト

curl -X POST -H "Content-Type: application/json" -d '{"id": 100, "name": Alex, "score": 96}' https://www.example.com/post

host

名前解決を検証し、簡易的な情報を出力するためのコマンド。

host [オプション] <名前>

#www.example.comを名前解決する

host www.example.com

ip

IPアドレスなどの設定情報を確認するためのコマンド。

ip [オプション] <オブジェクト> <サブコマンド>

#IPアドレスの確認

ip addr

#enp0s3のインターフェイスのみを確認する

ip addr show enp0s3

#ルーティングテーブルを表示する

ip route

nmcli

NetworkManagerを利用したネットワーク機能を確認、設定するためのコマンド。

nmcli [オブジェクト] <サブコマンド> <引数>

#接続情報を確認する

nmcli connection show

#enp0s3の接続情報を確認する

nmcli connection show enp0s3

ping

宛先となるホストと疎通確認を行うためのコマンド。

ping [オプション] <宛先IPアドレス>

#192.0.0.10に疎通確認を行う

ping 192.0.0.10

traceroute

ホストまでの経路を表示するためのコマンド。

traceroute [オプション] <ホスト名>

#googleまでの経路を表示

traceroute google.com
