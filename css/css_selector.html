CSS

セレクタ

2023/05/21

このページの説明

::after

::afterは要素のテキストの直後を選択するセレクタ

::after

::before

::beforeは要素のテキストの直前を選択するセレクタ

::before

::first-letter

::first-letterはテキストの最初の1文字を選択するセレクタ

::first-letter

::first-line

::first-lineは段落の1行目だけを選択するセレクタ

::first-line

::-moz-selection

::-moz-selectionはユーザが選択したテキストをを選択するセレクタ(Firefox専用selection)

::-moz-selection

::selection

::selectionはユーザが選択したテキストを選択するセレクタ

::selection

:active

:activeはリンクをクリックしたときのスタイルを変更するセレクタ

:active

:empty

:emptyは子要素がない、空の要素を選択するセレクタ

:empty

:focus

:focusはフォーム部品がフォーカスしたときにスタイルを適用するセレクタ

:focus

:hover

:hoverはリンクにマウスがホバーした状態のときのスタイルを変更するセレクタ

:hover

:invalid

:invalidは入力内容が誤っているときのスタイルを選択するセレクタ

:invalid

:last-child

:last-childは最後の要素を選択するセレクタ

:last-child

:link

:linkはリンク先が未訪問の<a>タグのスタイルを変更するセレクタ

:link

:nth-child(n)

:nth-child(n)はn番目の要素(リストなどのn番目の要素)を選択するセレクタ

:nth-child(n)

:required

:requiredは入力必須項目に入力がないときのスタイルを選択するセレクタ

:required

:target

:targetはクリックしたリンクのリンク先要素を選択するセレクタ

:target

:valid

:validは入力内容が正しいときのスタイルを選択するセレクタ

:valid

:visited

:visitedはリンク先が訪問済みの<a>タグのスタイルを変更するセレクタ

:visited
