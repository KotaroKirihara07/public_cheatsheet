このページの説明

text-align：

内包する要素に対して横位置の揃え方を指定する

text-align: 行揃え;
行揃え 説明
left 左
center 中央
right 右
justify 両端揃え

text-decoration：

文字列に装飾線や点滅を指定する

text-decoration: 線を引く位置;
線を引く位置 説明
overline 上
underline 下
line-through テキスト上
none なし

text-indent：

文章のインデント幅を指定する

text-indent: 幅;

text-overflow：

はみ出たテキストを扱いを指定する

text-overflow: 表示方法;
表示方法 説明
ellipsis ・・・
clip なし

quotes：

引用文の前後につけるかっこを定義する

quotes: "開始かっこ" "終了かっこ";

white-space：

改行を制御する

white-space: キーワード;
