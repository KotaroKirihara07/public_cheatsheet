setting.py

2024/08/04

このページではsetting.pyの編集方法について説明する。

アプリケーションの登録

INSTALLED_APPS = [
     'django.contrib.admin',
     'django.contrib.auth',
     'django.contrib.contenttypes',
     'django.contrib.sessions',
     'django.contrib.messages',
     'django.contrib.staticfiles',
    
     #アプリケーションを追加する (アプリケーションディレクトリ名を追加する)
     'app_name1',
     'app_name2',
]

データベースの設定

SQLite

DATABASES = {
     'default': {
         'ENGINE': 'django.db.backends.sqlite3',
         'NAME': BASE_DIR / 'db.sqlite3',
     }
}

PostgreSQL

DATABASES = {
     'default': {
         'ENGINE': 'django.db.backends.postgresql',
         'NAME': 'データベース名',
         'USER': 'ユーザ名',
         'PASSWORD': 'パスワード',
         'HOST': 'ホスト名',
         'PORT': 'ポート番号',
     }
}
