Docker

コンテナ

2025/07/13

このページではdocker container についての説明を行う。

docker container <command>

docker container run

docker container run はイメージをもとにコンテナを実行するためのコマンド。Docker Composeを使用することで代替可能。

docker container run [option] <image_name>

Dockerに関する情報を取得するためのコマンド

command 説明
info Dockerのシステムとホストに関する情報を表示する
help ヘルプ情報を表示する
version Docker Engineおよびコマンドのバージョンを表示する
