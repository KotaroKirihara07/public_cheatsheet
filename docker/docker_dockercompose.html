Docker Compose

2025/07/12

このページではDocker Composeについての説明を行う。

Docker Composeとはコンテナの構築に関わるコマンド文の内容をYAMLファイルに書き込んで、一気に実行したり、停止したり、破棄したりすることを可能にするもの。docker container run コマンドやdocker container run コマンドなどで実行する処理を代替可能。

docker-compose.ymlの設定項目

Compose file reference

#大項目

項目 説明
version Docker Composeのバージョンを指定する
services コンテナに関する定義をする
networks ネットワークに関する定義をする
volumes ボリュームに関する定義をする
configs
secrets

#services内の設定項目

項目 説明
image 利用するイメージを指定する
networks 接続するネットワークを指定する
volumes 記憶領域のマウントを設定する
ports ポートのマッピングを設定する
environment 環境変数を設定する
depends_on 別のサービスに依存することを示す
restart コンテナが停止したときの再試行ポリシーを設定する

docker compose

docker-composeは設定を記述したYAMLファイルをもとにコンテナを操作するためのコマンド。

docker-compose -f docker-compose.yml <command> [option]
command 説明
up
build
ps
run
logs
stop
rm
