VBE

マクロVBAの編集画面をVBE(Visual Basic Editor)という。VBEの開き方は以下の2通り。

  • 「開発」→「Visual Basic」
  • Alt + F11

標準モジュール

標準モジュールとは、マクロVBAを書く場所である。

#標準モジュールの作成 : 「挿入」→「標準モジュール」

#標準モジュールの名称変更 :「表示」→「プロパティウィンドウ」

#標準モジュールの削除: MODULE_NAMEを右クリック→「MODULE_NAMEの解放」

マクロの実行方法

マクロの実行方法は以下の3通り。

  • VBEの画面で実行
    • 「実行」→「Sub/ユーザフォームの実行」
    • F5
  • ワークシートの開発タブで実行
    • 「開発」タブ →「マクロ」 → マクロ名を選択 → 「実行」
    • Alt + F8
  • ボタンや図形に登録して実行
    • 「開発」タブ→「挿入」→ ボタン（フォームコントロール）→ 右クリック→「マクロの登録」 → クリックして実行
    • 「挿入」タブ→「図形」→ 好みの図形を選択 → 右クリック→「マクロの登録」→ クリックして実行
