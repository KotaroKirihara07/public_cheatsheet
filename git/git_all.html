Git

Git コマンド集

2024/11/14

このページではGitの基本的なコマンドをまとめています。以下がそれぞれのコマンドへのリンクです。

add, branch, clone, commit, config, help, init, log, merge, pull, push, remote, revert, status

git add

git addは更新内容をインデックスに登録するためのコマンド

git add [オプション] <ファイル/ディレクトリ>

#更新内容をインデックスに登録する(一つのファイル)

git add filename

#更新内容をインデックスに登録する(すべて)

git add .

#git addで何が実行されるか調べる

git add --dry-run

git branch

git branchはブランチを作成/表示するためのコマンド

git branch [オプション] <ブランチ名>

#branchのリストを表示する

git branch

#branch_nameという名前のbranchの作成

git branch branch_name

#branchの削除

git branch -d branch_name

git clone

git cloneはリポジトリの複製を取得するためのコマンド

git clone [オプション] <リポジトリ> <ディレクトリ>

#clone

git clone クローン元 クローン先

#ベアリポジトリを作成する

git clone --bare クローン元 クローン先

#GitHubからのクローン作成

git clone https://github.com/---

git commit

git commitは更新内容をコミットするためのコマンド

git commit [オプション]

#コミット

git commit -m "www"

#add and commit

git commit -a -m "www"

git config

git configはGitの初期設定を実行するためのコマンド

git config [オプション] <設定項目> <値>

#ユーザ名(user1)を設定する

git config --global user.name user1

#メールアドレスを設定する

git config --global user.email aaa@example.com

#設定内容を表示する

git config --global -list

git help

git helpはコマンドの使い方を表示するためのコマンド

git help [オプション] <サブコマンド>

#git commitの使い方を表示する

git help commit

#利用できるサブコマンドを表示する

git help -a

git init

git initはリポジトリを作成するためのコマンド

git init [オプション]

#リポジトリの作成

git init

git log

git logはコミットログを確認するためのコマンド

git log [オプション] <ファイル/ブランチ>

#コミットログを確認する

git log

git merge

git mergeはブランチをマージするためのコマンド

git merge [オプション] <ブランチ名>

#統合する

git merge branch_name

#2つの競合するブランチ間のマージを中止する

git merge -abort

git pull

git pullはリモートリポジトリ上の変更内容を取得するためのコマンド

git pull [オプション] <名前> <ブランチ>

#変更内容を取得する

git pull origin master

git push

git pushはローカルリポジトリの内容をリモートリポジトリに送信するためのコマンド

git push [オプション] <名前> <ブランチ>

#リモートリポジトリ(origin)にmasterにブランチの内容を送信する

git push origin master

#remote先にbranchを作る

git push --set-upstream origin new_branch

git remote

git remoteはリモートリポジトリを制御するためのコマンド

git remote [オプション] <サブコマンド>

#リモートリポジトリを表示する

git remote -v

#リモートリポジトリを追加する

git remote add 名前 URL

#リモートリポジトリを削除する

git remote remove 名前

#remoteのoriginの改名

git remote rename origin new_name

git revert

git revertはコミットした内容を取り消すためのコマンド

git revert [オプション] SHA1-ID

#SHA1-IDがaaaのコミットを取り消す

git revert aaa

git status

git statusは ワーキングツリーの状態を確認するためのコマンド

git status [オプション]

#状態を確認する

git status
