java FX

2023/12/10

javaFXとはリッチインターネットアプリケーションを作成するためのGUIライブラリである。FXMLとCSSを併用してデザインを記述する。https://zenn.dev/uinoue/articles/cb93c2d50f9c28

概要

ステージ内にシーン。シーン内にペイン。ペインをルートノードとして複数のコントロールがリーフノードして連結している。

ステージ

画面表示されるウインドウになる最上位のJavaFXコンテナ。ステージはウインドウに対応するクラスで、1つのアプリケーションで複数のステージを開くことができます。なお、アプリケーション起動時にプライマリステージが生成されるので、1つのウインドウだけで処理が完結する場合は新たなステージを生成する必要はありません。

Stageクラス

#import宣言

javafx.stage.Stage;

#新しいウインドウを作成する

Stage new_Stage = new Stage();

#メソッド集

メソッド 説明 戻り値の型
setScene(Scene value) ウインドウに表示するシーンを設定する void
setTitle(String text) 指定したテキストをウインドウのタイトルに設定する void
show() ウインドウを表示する void
close() ウインドウを閉じる void
hide() ウインドウを閉じる void

シーン

ステージに表示するすべての内容を木構造で保持するコンテナ。シーンは、ステージに表示する内容をシーングラフと呼ばれる木構造で保持します。１つのステージに表示できるシーンは１つだけですが、別のシーンに切り替えることが可能です。

Sceneクラス

#import宣言

javafx.scene.Scene;

#シーンを作成する

Scene scene = new Scene(vbox);

ペイン

表示するノードのレイアウトを管理するコンテナ。ペインはレイアウトペインとも呼ばれ、表示するノードのレイアウトを制御します。なお、ノードはシーングラフの構成要素となるもので、ペインとコントロールはノードの下位クラスです。１つのシーンで複数のペインを使用できますが、木構造のルートとなるペインは１つだけです。

VBoxクラス

#import宣言

javafx.scene.layout.VBox;

#VBoxペインを作成し、作成した部品を配置する

VBox vbox = new VBox(label, textField, button);

HBoxクラス

#import宣言

javafx.scene.layout.HBox;

#HBoxペインを作成し、作成した部品を配置する

HBox hbox = new HBox(label, textField, button);

コントロール

ユーザが画面上で操作できるノード。コントロールは、エンドユーザが画面上で認識して、マウスやキーボードで操作できる部品の総称です。ラベル、ボタン、テキストフィールド、ポップアップリスト、メニューなど、非常に多くの部品があります。１つのペインに複数のコントロールを配置できます。

Labelクラス

#import宣言

import javafx.scene.control.Label;

#ラベルを作成する

Label label = new Label("ラベル名");

#指定したテキストをラベルに設定する

label.setText(String text);

Buttonクラス

#import宣言

import javafx.scene.control.Button;

#ボタンを作成する

Button button = new Button("ボタン名");

#メソッド集

メソッド 説明 戻り値の型
setText(String text) 指定したテキストをボタンに表示する void
setOnAction(EventHandler<ActionEvent> value) ボタンがクリックされた際に実行するアクションを設定する void

#ActionEvent


TextFieldクラス

#import宣言

import javafx.scene.control.TextField;

#テキストフィールドを作成する

TextField textField = new TextField("ここに入力してください");;

#メソッド集

メソッド 説明 戻り値の型
textField.getText(); テキストフィールドに入力されたテキストを取得する String
textField.setText(String text); 指定したテキストでテキストフィールドを上書きする void
setPrefColumnCount(int value); テキストフィールドの優先列数（領域の表示幅を決めるための基準値）を設定する void

イベント

主にユーザの操作によって発生するイベント。エンドユーザがマウスでボタンをクリックしたり、キーボードから文字を入力したりするとイベントが発生します。ボタンやテキストフィールドにイベントハンドラと呼ばれるメソッドを定義しておくことにより、ユーザは発生したイベントを自由に処理できます。
