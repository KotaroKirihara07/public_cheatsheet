Systemクラス 、 Stringクラス 、 Mathクラス

インスタンス化の必要がない。

Systemクラス

出力を扱うためのクラス。

#出力 (改行あり)

System.out.println(12345);

#出力 (改行なし)

System.out.print(12345);

Stringクラス

文字列を扱うためのクラス。

#文字列の生成

String str = "abcde";
var str = new String("abcde");

#文字列への変換

String str = String.valueOf(123);

#メソッド集

メソッド 説明 戻り値の型
str.codePointCount(0, str.length()) 文字列の長さを調べる int
str.isEmpty() 文字列が空であるかどうか boolean
str.isBlank() 空文字列または空白のみかどうか boolean
str.indexOf(substr) 特定の文字列が最初に登場する文字列のインデックスを返す (存在しなければ-1を返す) int
str.indexOf(substr, n) 特定の文字列が最初に登場する文字列のインデックスを返す (検索開始位置をnとする, 存在しなければ-1を返す) int
str.lastIndexOf(substr) 特定の文字列が最後に登場する文字列のインデックスを返す (存在しなければ-1を返す) int
str.contains(substr) 文字列を含むかどうか boolean
str.startsWith(substr) str先頭の文字列がsubstrと一致するかどうか boolean
str.endsWith(substr) str末尾の文字列がsubstrと一致するかどうか boolean
str.compareTo(substr) 文字列が一致しているかどうか (0で一致) int
str.compareToIgnoreCase(substr) 大文字小文字区別しないで文字列が一致しているかどうか (0で一致) int
str.equals(substr) 文字列が一致しているかどうか (true or false) boolean
str.equalsIgnoreCase(substr) 大文字小文字区別しないで文字列が一致しているかどうか boolean
str.strip() 文字列の前後の空白を除去する void
str.stripLeading() 文字列の前部の空白を除去する void
str.stripTrailing() 文字列の後部の空白を除去する void
str.substring(2, 5) 部分文字列を取得する String
str.split(",") 特定の区切り文字で分割する (カンマ区切り) String
str.replaceAll(a, b) strの"a"をすべて"b"に変換する。(第一引数を正規表現にすることも可能) void

#クラスメソッド集

メソッド 説明 戻り値の型
String.join(",", str) 特定の区切り文字で連結する (カンマで連結) String
String.format("%sは%d歳です。", "私", 20) 文字列に値を埋め込んで返す String

#変換文字

変換文字 説明
%b, %B ブール値
%h, %H ハッシュコードを16進数に変換
%s, %S 文字列
%c, %C Unicode文字
%d 10進数
%o 8進数
%x, %X 16進数
%e, %E 浮動小数点数形式の10進数
%f 10進小数点数
%t, %T 日付、時刻変換文字の接頭辞
%% '%'
%n プラットフォーム固有の行区切り文字

Mathクラス

数学的演算を行うためのクラス。

#クラスメソッド集

メソッド 説明 戻り値の型
Math.abs(x) 絶対値 var
Math.min(a,b) 最小値 var
Math.max(a,b) 最大値 var
Math.ceil(x) 数値の切り上げ double
Math.floor(x) 数値の切り捨て double
Math.round(x) 数値を四捨五入 int
Math.sqrt(x) 平方根 double
Math.cbrt(x) 立方根 double
Math.pow(x,y) xのy乗 double
Math.sin(x) サイン double
Math.sin(x) サイン double
Math.cos(x) コサイン double
Math.tan(x) タンジェント double
Math.asin(x) アークサイン double
Math.acos(x) アークコサイン double
Math.atan(x) アークタンジェント double
Math.toRadians(x) 角度をラジアンに変換 double
Math.toDegrees(x) ラジアンを度に変換 double
Math.exp(x) eのx乗 double
Math.log(x) 自然対数 double
Math.log10(x) 底が10の対数 double
