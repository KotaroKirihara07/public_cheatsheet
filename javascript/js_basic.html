データの型

データはプリミティブ型とオブジェクト型の2つがある。プリミティブ型のデータは不変であるのに対して、とオブジェクト型の変数は可変である 。

#データの型を調べる

typeof var1;

#オブジェクトのインスタンスかどうかを調べる (true or false)

suzuki instanceof Person;

#プリミティブ型

データ型 説明 例
Boolean 真偽値 (プリミティブ型) true, false
String 文字列 (プリミティブ型) 'aaa', "bbb"
Number 数値 (プリミティブ型) 100
BigInt 巨大な整数 (プリミティブ型) 12345678910h
undefined 未定義 (プリミティブ型) undefined
null ヌル (プリミティブ型) null
Symbol シンボル (プリミティブ型) Symbol
Object オブジェクト (非プリミティブ型) Array,Object,Date等のプリミティブ型以外のすべて

#型の変換

メソッド 説明 戻り値
Boolean(値) 値を真偽値に変換する 真偽値
String(値) 値を文字列に変換する 文字列
Number(値) 値を数値型に変換する 数値
parseInt(文字列) 文字列を数値型(整数)に変換する 数値
parseFloat(文字列) 文字列を数値型(浮動小数点)に変換する 数値

配列

#配列を定義する

let array1 = [];
const array2 = [1,2,3];
const array3 = ['a','b'];
const array4 = [[1,2,3],[4,5,6]];
let array5 = [{id: 1, name: 'a'},{id:2, name: 'b'}];

#配列の各要素に対して処理をする

array1.forEach((value,index) => {
  console.log(index,value);
});

#配列から条件を満たす要素を取得する

const target_element = array1.find((element) => element == "aaa");

#配列から条件を満たす要素の最初のインデックスを取得する

const target_index = array1.findIndex((element) => element == "aaa");

#ある配列から別の配列を作成する

const Id_list = array1.map((value,index) => ${index}_${value});

#ある配列から条件を満たす別の配列を作成する

let array2 = array1.filter((value) => value >= 10);

#メソッド集

メソッド 説明 戻り値
array1.length; 配列の長さを取得する 数値
array1.unshift(var1,var2,...) 先頭に要素を追加する 追加後の個数
array1.push(var1,var2,...) 末尾に要素を追加する 追加後の個数
array1.shift() 先頭の要素を取り除く 取り除かれた要素
arrary1.pop() 末尾の要素を取り除く 取り除かれた要素
array1.splice(追加位置,取り出す数,var1, var2,...) 指定位置から要素を取り出し、要素を追加する 配列
arrary1.concat(array2,array3,...) 配列1に配列2,配列3を結合する なし
array1.join('') 配列の各要素を結合して文字列にする 文字列
array1.indexOf(var1) 要素のインデックスを調べる 数値
array1.lastindexOf(var1) 要素の末尾からのインデックスを調べる 数値
array1.includes(var1) 要素が含まれているかどうか 真偽値
array1.reverse() 配列の並び順を反転する 配列

#オブジェクトの定義

const object1 = {};

const var1 = object2[id];

#オブジェクトのプロパティの値を更新する

#オブジェクトのプロパティを削除する

#オブジェクトのメソッドを上書きする

object.func = function() { //処理 };
object["func"] = function() { //処理 };

Object.values(object1) オブジェクトの各値の配列 配列
Object.entries(object1) オブジェクトの各プロパティの配列 配列

for (let index=0; index < 10; index++){
  処理;
  if (index == 5) {
    continue;
  }
}

for (const value of array){
  処理;
}

#for文 (オブジェクト用)

for (const key in object1){
  処理;
}

#while文

while (var1 < 10) {

条件分岐

#if文

if (var1 >= 10) {
  処理1;
} else if (var1 <= 5){
  処理2;
} else {
  処理3;
}

#switch文 (var1の値が1のとき処理1、2のとき処理2、その他は処理3)

switch (var1) {
  case 1:
    処理1;
    break;
  case 2:
    処理2;
    break;
  default:
    処理3;
    break;
}

クラス

#クラスの定義1

class MyClass {
  constructor() {
    console.log("a");
    console.log("b");
    this.value1 = 100;
  }
  Mymethod1() {
    console.log("www");
  }
  //スタティックメソッド
  static Mymethod2() {
    this.Mymethod1();
    console.log("static method");
  }
}

#クラスの定義2 (継承)

class Childclass extends MyClass {
  constructor() {
    super();
  }
  Mymethod3() {
    console.log("???");
  }
}

#クラスの定義3

class MyClass {
  //スタティックフィールド
  static STATIC_VALUE = "static value";

  //セッター
  set name(value) {
    this.name = value;
  }

  //ゲッター
  get name() {
    return this.name;
  }
}

#インスタンス化

const myclass = new Myclass();

#スタティックフィールドを呼び出す

const st_value = Myclass.STATIC_VALUE;

#スタティックメソッドを呼び出す

Map

#Mapオブジェクトを初期化する

const map_list = new Map()

#メソッド集

メソッド 説明 戻り値
map_list.set(key,value) キーと値のセットをマップに登録する Mapオブジェクト
map_list.get(key) キーを指定して値を取り出す 値
map_list.has(key) キーの値が存在するか 真偽値
map_list.delete(key) キーの値を削除する 真偽値(削除成功かどうか)
map_list.clear() キーと値をすべて削除する なし
map_list.keys() キーからなるIteratorオブジェクト Iteratorオブジェクト
map_list.values() 値からなるIteratorオブジェクト Iteratorオブジェクト
map_list.entries() キーと値の配列からなるIteratorオブジェクト Iteratorオブジェクト

#キーと値のペア数

let length = map_list.size

#各ペアに対して処理を行う

map_list.forEach((value,key) => {
  console.log(${key} : ${value});
})

Set

Setオブジェクトを初期化する

const set_list = new Set()

#メソッド集

メソッド 説明 戻り値
set_list.add(value) 値を登録する Setオブジェクト
set_list.has(value) 値が存在するか 真偽値
set_list.delete(value) 値を削除する 真偽値(削除成功かどうか)

set_list.keys() 各値を返す Iteratorオブジェクト

#要素の数

let length = set_list.size

set_list.forEach((value) => {

Error

#Errorインスタンスを生成する

const error = new Error('エラーが発生しました');

#エラー内容を取得する

const er_string = error.message;

#エラーを投げる

throw error;

#try-catch-finally構文

try {
  #通常時の処理
} catch (error) {
  #エラー時の処理
} finally {
  #エラーの有無に関わらずに実行する処理
}

#メソッド集

メソッド 説明 戻り値
console.log(var1,var2,..) ログを表示する なし
console.info(var1,var2,..) 情報を表示する なし
console.warn(var1,var2,..) 警告を表示する なし
console.error(var1,var2,..) エラーを表示する なし
console.dir(オブジェクト) オブジェクトの構造を出力する なし
console.table(オブジェクト) オブジェクトの構造を表組みで表示する なし

#エラーの種類

エラーの種類 説明
RangeError 値が許容範囲にない
ReferenceError 宣言されていない変数を読みだそうとした
SyntaxError 言語の構文が不正
TypeError データ型が不正
URIError URIが不正
