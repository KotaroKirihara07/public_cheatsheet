account

このページの説明

ユーザ系 : useradd、usermod、userdel、passwd、id

グループ系 : groupadd、groupmod、groupdel

アカウントの設定ファイル

ファイル 説明
/etc/passwd ユーザ情報が保存されているファイル
/etc/shadow パスワード情報が保存されているファイル
/etc/group グループ情報が保存されているファイル

useradd

ユーザを追加するためのコマンド。

useradd [option] <user_name>

option 説明
-u <UID> UID(ユーザID)を指定する
-g <group_name>/<GID> 主グループ名を指定する
-G <group_name>/<GID> 補助グループ名を指定する
-m ホームディレクトリを作成する

usermod

ユーザの設定を編集するためのコマンド。

usermod [option] <user_name>

option 説明
-u <UID> UID(ユーザID)を変更する
-g <group_name>/<GID> 主グループ名を変更する
-G <group_name>/<GID> 補助グループ名を変更する
-l <user_name> ユーザ名を変更する

userdel

ユーザを削除するためのコマンド。

userdel [option] <user_name>

option 説明
-r ホームディレクトリも含めて削除する
-f ログイン中のユーザを削除する

passwd

ユーザにパスワードを設定するためのコマンド。

passwd [option] <user_name>

option 説明
-l アカウントをロックする
-u アカウントのロックを解除する

id

ユーザの情報を表示するためのコマンド。

id [option] <user_name>

option 説明
-u ユーザIDを表示する
-g グループIDを表示する
-G 補助グループを含むグループIDを表示する
-n ユーザID、グループID、補助グループIDを表示する

groupadd

グループを作成するためのコマンド。

groupadd [option] <group_name>

option 説明
-g <GID> グループIDを指定する

groupmod

グループの設定を編集するためのコマンド。

groupmod [option] <group_name>

option 説明
-g <GID> グループIDを変更する
-n <group_name> グループ名を変更する

groupdel

グループを削除するためのコマンド。

groupdel <group_name>
