ディストリビューション

主なLinuxのディストリビューションは「Red Hat系」、「Debian系」の2種類存在する。

Red Hat系

  • Red Hat Enterprise Linux
  • Cent OS / Cent OS Stream
  • Fedora
  • Alma Linux

Debian系

  • Debian GNU/Linux
  • Ubuntu

ユーザ

Linuxにおけるユーザは「rootユーザ」、「一般ユーザ」に分けられる。

rootユーザ

システム管理が可能であるユーザ。

一般ユーザ

操作に制限があるユーザ。

ディレクトリ構造

ディレクトリ名 用途
/etc 設定ファイル
/home 一般ユーザ用のホームディレクトリ
/root rootユーザ用のホームディレクトリ
/boot 起動に関するファイル
/bin 一般ユーザが実行可能なコマンドの実行ファイル
/sbin rootユーザのみが実行可能なコマンドの実行ファイル
/lib ライブラリファイル
/proc プロセスやカーネルについての情報
/dev デバイスファイル
/usr インストールされたプログラムやコマンドに関するファイル
/var ログファイルやWeb公開用のドキュメントなど
/tmp 一時的なファイル
/mnt ファイルシステム用のマウントポイント
/media メディアファイルのマウントポイント
