基本

#リダイレクトによる出力先の指定

構文 説明
コマンド > ファイル名 標準出力をファイルに上書きする
コマンド >> ファイル名 標準出力をファイルに追記する
コマンド &> ファイル名 コマンド実行時のすべての出力結果をファイルに上書きする
コマンド &>> ファイル名 コマンド実行時のすべての出力結果をファイルに追記する

awk

空白やカンマなどで区切られたテキストを列として扱い、出力したり特定の処理をしたりするためのコマンド。

awk [オプション] '<パターン>{アクション}' <ファイル名>

cat

ファイルの内容を表示するコマンド。

cat [オプション] <ファイル名>

オプション 説明
-n 行番号を添えて表示する

cd

ディレクトリを変更するためのコマンド。

cd <ディレクトリ名>

cp

ファイル、ディレクトリをコピーするためのコマンド。

cd [オプション] <コピー元ファイル/ディレクトリ> <コピー先ファイル/ディレクトリ>

オプション 説明
-r ディレクトリをコピーする

cut

入力行の一部を切り出して出力するコマンド。

cut -d <区切り文字> -f <フィールド番号> [ファイル名]

#「:」で区切り, 7番目のフィールドを切り出して表示

cut -d : -f 7 file1

diff

ファイルの差分を表示するためのコマンド。

diff [オプション] <比較元ファイル> <比較先ファイル>

find

ファイル探索のためのコマンド。

find <検索ディレクトリ> [オプション] <検索条件> <アクション>

#現在のディレクトリ内のファイルの探索してファイル名を表示

find . -name file.txt -ls

#現在のディレクトリ内の特定の拡張しを持つファイルを探索してファイル名を表示

find . -name *.txt -ls

gedit

ファイルを編集するためのコマンド。

gedit <ファイル名>

grep

ファイル内の文字列を検索するコマンド。検索パターンに正規表現を使用可能。

grep [オプション] <検索パターン> <ファイル名>

オプション 説明
-e 複数のキーワードを指定する
-v 否定条件
-i 大文字、小文字の区別をつけない
-c キーワードと合致する行数を表示する
-n 該当行番号を出力する

head

ファイル内の先頭数行を表示するためのコマンド。

head [オプション] <ファイル名>

less

ファイルの内容をスクロール表示するためのコマンド。

less [オプション] <ファイル名>

ls

ファイル名やディレクトリ名を表示するためのコマンド。

ls [オプション] [ディレクトリ/ファイル]

オプション 説明
-a 隠しファイルを含むすべてのディレクトリ/ファイルを表示
-F ファイル種別を表示する
-l ファイル名以外の詳細な情報を表示する
-R サブディレクトリも含めて表示する

mkdir

ディレクトリを作成するためのコマンド。

mkdir [オプション] <作成するディレクトリ>

オプション 説明
-p 途中のディレクトリとともに新しいディレクトリを作成

mv

ファイル、ディレクトリ移動のためのコマンド。 ファイルの変更も可能。

mv [オプション] <移動元ファイル/ディレクトリ> <移動先ファイル/ディレクトリ>

pwd

現在のディレクトリを表示するためのコマンド。

pwd

rm

ファイル,ディレクトリ削除のためのコマンド。

rm [オプション] <削除するファイル/ディレクトリ> <削除するファイル/ディレクトリ> ...

オプション 説明
-r ディレクトリを削除
-i 確認後にファイルを削除

rmdir

空のディレクトリを削除するためのコマンド。

rmdir <ディレクトリ名>

sed

文字列を置換するためのコマンド。

sed [オプション] <スクリプト> <対象ファイル>

sort

ファイルの内容をソートするコマンド。

sort [オプション] <ファイル名>

オプション 説明
-n 数値順に並べる
-r 逆順に並べる

tail

末尾部分を表示するコマンド。

tail [オプション] <ファイル名>

tar

アーカイブ作成のためのコマンド。

tar [オプション] <ファイル/ディレクトリ>

オプション 説明
-c アーカイブファイルを作成する
-x アーカイブファイルを展開する
-t アーカイブファイルの内容を参照する
-x アーカイブファイルを展開する
-c アーカイブファイルを作成する
-f <ファイル名> アーカイブファイル名を指定する
-C <ディレクトリ名> 展開先ディレクトリを指定する

touch

空のファイルを作成するためのコマンド。

touch <新しいファイル名1> <新しいファイル名2> ...

tree

ディレクトリの構成をツリー状に表示するためのコマンド。

tree [オプション] [ディレクトリ]

オプション 説明
-a すべてのディレクトリ/ファイルを表示する
-d ディレクトリのみ表示する

unique

重複を除いたファイルの内容を返すコマンド。

unique [オプション] <ファイル名>

オプション 説明
-c 重複行の数を表示する

unzip

zipファイルを展開するためのコマンド。

unzip [オプション] <ファイル.zip>

オプション 説明
-d ディレクトリ 展開先のディレクトリを指定する
-l zipファイルの内容を表示する

wc

ファイルの行数、単語数、バイト数を数えるためのコマンド。

wc [オプション] <ファイル名>

オプション 説明
-l 行数を表示する
-c ファイルの容量を表示する
-w 単語数を表示する

zip

ファイルをアーカイブ、圧縮するためのコマンド。

zip [オプション] <ファイル.zip> <圧縮対象ファイル/ディレクトリ>

オプション 説明
-r ディレクトリ内のファイルを再帰的に処理する
-e パスワードを指定して暗号化されたzipファイルを作成する
