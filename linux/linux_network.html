curl

指定したURL宛てにWeb接続を行うためのコマンド。

curl [option] <URL>

#AWS CloudShellのグローバルIPアドレスを取得する

curl http://checkip.amazonaws.com/

#JSON形式のリクエストボディを使用したPOSTリクエスト

curl -X POST -H "Content-Type: application/json" -d '{"id": 100, "name": Alex, "score": 96}' https://www.example.com/post

host

名前解決を検証し、簡易的な情報を出力するためのコマンド。

host [option] <name> <DNS_server>

option 説明
-t <record> 問い合わせるレコードの種類を指定する
-v 詳細な情報を表示する

ip

IPアドレスなどの設定情報を確認するためのコマンド。

ip [option] <object> <sub_command>

object sub_command 説明
address - IPアドレスを表示する
address show <interface_name> 設定情報を表示する
route - ルーティングテーブルを表示する

nmcli

NetworkManagerを利用したネットワーク機能を確認、設定するためのコマンド。

nmcli <object> <sub_command> <args>

object 説明
connection 接続情報
general Network Mangerサービスの管理
device デバイスの管理

sub_command 説明
show 設定を参照する
modify 設定を変更する
up 接続を有効化する
add 接続プロファイルを追加する

ping

宛先となるホストと疎通確認を行うためのコマンド。

ping [option] <IP_address>

option 説明
-c <N> N回パケットを送信する

traceroute

ホストまでの経路を表示するためのコマンド。

traceroute [option] <URL>
