このページの説明

Red Hat系 : rpm、yum

Debian系 : dpkg、apt

rpm

rpmパッケージを管理するためのコマンド。

rpm [option] <package_name>/<file_name>

option 説明
-i インストール
-U アップグレードする (インストールされていない場合はインストールする)
-F アップグレードする (インストールされていない場合はインストールしない)
-e アンインストール
-q パッケージを検索する
-qa インストール済みのすべてのパッケージを表示する

yum

リポジトリ経由でrpmパッケージを操作する

yum [option] <sub_command> <package_name>

option 説明
-y 確認ダイアログにすべて「y」と回答する

sub_command 説明
install パッケージをインストール/更新する
check-update 更新パッケージの有無をチェックする
update パッケージを更新する
remove パッケージを削除する
info パッケージを検索する
list パッケージ一覧を表示する

#yumの設定ファイル

file 説明
/etc/yum.conf yumの全般的な設定ファイル
/etc/yum.repos.d/* 各リポジトリの設定ファイル

dpkg

debパッケージを管理するためのコマンド。

dpkg [option] <package_name>/<file_name>

option 説明
-i インストール
-r アンインストール
-p 設定ファイルも含めてアンインストール
-l インストール済みパッケージを検索する
-L インストールされたファイル一覧を表示する
-s パッケージの情報を表示する
-S ファイルのインストール元パッケージを検索する

apt

リポジトリ経由でrpmパッケージを操作する

apt <sub_command> <package_name>

sub_command 説明
update パッケージ情報を更新する
install パッケージをインストール
update パッケージを更新する
remove パッケージを削除する
list パッケージ一覧を表示する
show パッケージ情報を表示する
search パッケージを検索する
