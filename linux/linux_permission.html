permission

2025/06/25

説明。

chgrp、chmod、chown、su、sudo

chgrp

ファイルやディレクトリの所有グループを変更するためのコマンド。

chgrp [option] <group_name> <file_name>/<directory_name>

option 説明
-R 指定したディレクトリ内のすべてのディレクトリとファイルの所有グループも変更する

chmod

ファイルやディレクトリの権限を変更するためのコマンド。

chmod [option] <mode> <file_name>/<directory_name>

option 説明
-R 指定したディレクトリ内のすべてのディレクトリとファイルの権限も変更する

# mode

権限変更対象者 : |所有者|所有グループ|その他|

権限 : 0 (権限なし)、1 (実行)、2 (書き込み)、4 (読み取り)

chown

ファイルやディレクトリの所有者を変更するためのコマンド。

chown [option] <user_name>[:<group_name>] <file_name>/<directory_name>

option 説明
-R 指定したディレクトリ内のすべてのディレクトリとファイルの所有者も変更する

su

ユーザを切り替えるためのコマンド。ユーザ名を指定しないとrootユーザに切り替わる。

su [option] <user_name>

sudo

rootユーザ権限でコマンドを実行するためのコマンド。

sudo [option] <command>
