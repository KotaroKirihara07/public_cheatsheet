scp

SSHによりリモートホスト上にまたはリモートホスト上からファイルをコピーするためのコマンド。

scp [option] <file_path> <user_name@host_name:file_path>

option 説明
-P <port_number> 指定したポート番号で接続する
-r ディレクトリを再帰的にコピーする
-i <file_name> 秘密鍵ファイルを指定する

ssh

SSHによるリモート接続を行うためのコマンド。「/etc/ssh/sshd_config」はsshdの設定ファイル。

ssh [option] <user_name@host_name> <command>

option 説明
-p <port_number> 指定したポート番号で接続する
-l <user_name> ユーザ名を指定する
-i <file_name> 秘密鍵ファイルを指定する

#192.168.0.10のuser2にSSH接続する

ssh user2@192.168.0.10

#EC2インスタンスにSSH接続

ssh -i ".ssh/key-pair.pem" ec2-user@ec2-??-???-??-??.ap-northeast-1.compute.amazonaws.com

ssh-copy-id

SSh接続で利用する公開鍵を接続先にコピーして配置するためのコマンド。

ssh-copy-id [option] <user_name@host_name>

option 説明
-p <port_number> 指定したポート番号で接続する
-l <user_name> ユーザ名を指定する
-i <file_name> 秘密鍵ファイルを指定する

ssh-keygen

SSH接続で利用するキーペアを生成するためのコマンド。

ssh-keygen [option]

option 説明
-t <algorithm> 公開鍵ペアを生成するときに使用するアルゴリズムを指定する (デフォルトはrsa)
-b <N> Nビットの鍵を生成する
-f <file_path> ファイルパスを指定する

openssl

TLS/SSLプロトコルを実装するためのコマンド。

openssl [option]

#2048ビット秘密キーを作成する (非暗号化バージョン)

openssl genrsa 2048 | openssl pkcs8 -topk8 -v2 des3 -inform PEM -out ~/.config/rsa_key.p8

#2048ビット秘密キーを作成する (暗号化バージョン)

openssl genrsa 2048 | openssl pkcs8 -topk8 -v2 des3 -inform PEM -out ~/.config/rsa_key.p8

#公開キーを作成する

openssl rsa -in ~/.config/rsa_key.p8 -pubout -out ~/.config/rsa_key.pub
