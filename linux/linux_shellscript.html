ファイルの作成と実行

#ファイルの実行手順

  1. file_name.sh を作成する (ファイル内の文頭は #!/bin/bash) とする
  2. chmod a+x file_name.sh で実行権限を付与する
  3. bash file_name.sh で実行

#コマンドの連続実行

構文 説明
コマンド1 ; コマンド2 コマンド1の後コマンド2を連続で実行する
コマンド1 && コマンド2 コマンド1の実行が成功した場合、コマンド2を連続で実行する
コマンド1 || コマンド2 コマンド1の実行が失敗した場合、コマンド2を連続で実行する

変数

$var1

#四則演算

i=$(($1 + $2))
i=$(($1 - $2))
i=$(($1 * $2))
i=$(($1 / $2))
i=$(($1 % $2))

#特殊な変数

変数 説明
$0 スクリプトファイル名
$1, $2, $3 引数 ($1は1つ目の引数, $2は2つ目の引数)
$@ / $* すべての引数
$# 引数の数
$? 直前の処理の成否 (成功は0、失敗は0以外)
$$ シェルのPID
$RANDOM 0から32767の範囲の乱数を出力

条件分岐

#if文

if [条件式]
then
  処理
elif [条件式]
then
  処理
else
  処理
fi

#case式

case $var1 in
  値1) 処理 ;;
  値2) 処理 ;;
  *) 処理 ;;
esac

#条件式

条件式 説明
-f ファイル名 ファイルが存在するかどうか
-d ディレクトリ名 ディレクトリが存在するかどうか
-e ファイル名/ディレクトリ名 ファイル/ディレクトリが存在するかどうか
-r ファイル名/ディレクトリ名 ファイル/ディレクトリが読み取り可能かどうか
-w ファイル名/ディレクトリ名 ファイル/ディレクトリが書き込み可能かどうか
-x ファイル名/ディレクトリ名 ファイル/ディレクトリが実行可能かどうか
値1 -eq 値2 値1と値2が等しい
値1 -ne 値2 値1と値2が等しくない
値1 -lt 値2 値1が値2より小さい
値1 -le 値2 値1が値2以下
値1 -gt 値2 値1が値2より大きい
値1 -ge 値2 値1が値2以上
! 条件式 条件を満たさない (NOT)
条件式1 -a 条件式2 条件式1と条件式2の両方を満たす (AND)
条件式1 -o 条件式2 条件式1と条件式2のどちらかを満たす (OR)
-n 文字列 文字列の長さが0でない
