scp

SSHによりリモートホスト上にまたはリモートホスト上からファイルをコピーするためのコマンド。

scp [オプション] <コピー元> <コピー先>

#file.txtを192.168.0.10のホームディレクトリにコピーする

scp file.txt 192.168.0.10:~

#file1をdir1にコピーする

scp -i ".ssh/key-pair.pem" file1 ec2-user@ec2-??-???-??-???.ap-northeast-1.compute.amazonaws.com:~/dir1/

#dir1をdir2にコピーする

scp -i ".ssh/key-pair.pem" -r dir1 ec2-user@ec2-??-???-??-???.ap-northeast-1.compute.amazonaws.com:~/dir2/

ssh

SSHによるリモート接続を行うためのコマンド。

ssh [オプション] <ユーザ@>ホスト <コマンド>

#192.168.0.10にSSH接続する

ssh 192.168.0.10

#192.168.0.10のuser2にSSH接続する

ssh user2@192.168.0.10

#192.168.0.10にポート22を指定してSSH接続する

ssh -p 22 192.168.0.10

#EC2インスタンスにSSH接続

ssh -i ".ssh/key-pair.pem" ec2-user@ec2-??-???-??-??.ap-northeast-1.compute.amazonaws.com

ssh-copy-id

SSh接続で利用する公開鍵を接続先にコピーして配置するためのコマンド。

ssh-copy-id [オプション] <ユーザ名@>接続先

#生成した公開鍵を192.168.0.10にコピーして配置する

ssh-copy-id 192.168.0.10

#生成した公開鍵を192.168.0.10のユーザuser4にコピーして配置する

ssh-copy-id user4@192.168.0.10

ssh-keygen

SSH接続で利用するキーペアを生成するためのコマンド。

ssh-keygen [オプション]

#SSH接続で利用する2048ビットRSAのキーペアを生成する (秘密鍵: ~/.ssh/ssh-key、公開鍵: ~/.ssh/ssh-key.pub)

ssh-keygen -b 2048 -t rsa -f ~/.ssh/ssh-key

openssl

TLS/SSLプロトコルを実装するためのコマンド。

openssl [オプション]

#2048ビット秘密キーを作成する (非暗号化バージョン)

openssl genrsa 2048 | openssl pkcs8 -topk8 -v2 des3 -inform PEM -out ~/.config/rsa_key.p8

#2048ビット秘密キーを作成する (暗号化バージョン)

openssl genrsa 2048 | openssl pkcs8 -topk8 -v2 des3 -inform PEM -out ~/.config/rsa_key.p8

#公開キーを作成する

openssl rsa -in ~/.config/rsa_key.p8 -pubout -out ~/.config/rsa_key.pub
