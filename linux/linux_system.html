system

このページの説明

systemctl

systemユニットを制御するためのコマンド。

systemctl [comamnd] <unit_name>

コマンド 説明
start 起動
stop 停止
restart 再起動
status 状態の確認
enable 自動起動の有効化
disable 自動起動の無効化

ユニット名 説明
httpd Apache HTTP server
mysql MySQL

export

環境変数を設定する

export <ENV>=<value>

df

ディスク領域の使用状況やマウント状況を表示するためのコマンド。

df [option] <device_file>/<mount_point>

option 説明
-h MB/GB/TBを付けて表示する
-T ファイルシステムの種類も表示する

mount

マウント操作をするためのコマンド。/etc/fstabファイルに設定を保存すると起動時に自動マウントされる。

mount [option] <device_file> <mount_point>

option 説明
-a /etc/fstabでautoオプションが指定されているデバイスをすべてマウントする
-t <file_system> デバイスのファイルシステムを指定する

unmount

マウントを解除するためのコマンド。

unmount [option] <device_file>/<mount_point>

option 説明
-a マウントされているデバイスのすべてのマウントを解除する
-t <file_system> 指定したファイルシステムのデバイスのマウントを解除する

ln

リンクファイルを作成するためのコマンド。

ln [option] <from_link_file>/<to_link_file>

option 説明
-f リンク先を上書きする
-s シンボリックリンクを作成する

gdisk

GPTパーティションを作成するためのコマンド。

gdisk [option] <device_file>

option 説明
-l 指定したディスクのパーティション構成を表示する

mkfs

ファイルシステムを作成するためのコマンド。

mkfs [option] <device_file>

option 説明
-t <file_system> 指定したファイルシステムを作成する
-L ラベル名を指定する

file system 説明
ext2 Linuxで標準的に使われていたファイルシステム
ext3 ext2にジャーナリング機能を追加したファイルシステム
ext4 Debian 11の既定のファイルシステム
xfs CentOS 7の既定のファイルシステム
btrs スナップショット機能や複数デバイスを扱う機能をもつファイルシステム
vflat Windowsで利用されているファイルシステム
ntfs Windowsで利用されているファイルシステム
iso9660 CDやDVDなどのメディアで利用されるファイルシステム
tmpfs メモリ上に作成されるファイルシステム
