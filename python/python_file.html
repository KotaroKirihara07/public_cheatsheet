このページではファイルの取り扱い方法についての説明を行う。

基本

#ファイルの操作 (with命令を使用することでファイルのclose処理をしなくていい)

with open("data/example.txt", mode=モード, encoding="UTF-8") as f:
    #処理

#メソッド集

メソッド 説明 戻り値
f.read() テキスト内のすべてのデータをひとつの変数に格納する str
f.readline() 1行ごとのデータ str
f.readlines() データを行単位で分割したリストとして格納する list
f.write("wwww") 指定した文字列を書き込み、書き込まれた文字数を返す int
f.writelines(list1) 指定した文字列リストを1つずつ書き込む -

#オープンモード

モード 説明
"r" 読み込みモードで開く (デフォルト)
"r+" 読み書きモードで開く (デフォルト)
"a" 追記モードで開く
"a+" 読み込み+追記モードで開く
"w" 書き込みモード開く(ファイルが存在しなかった場合は新しく作成)
"w+" 読み書きモード開く(ファイルが存在しなかった場合は新しく作成)
"x" 書き込みモード開く(ファイルが存在する場合はエラー)
"x+" 読み書きモード開く(ファイルが存在する場合はエラー)
"b" バイナリモード
"t" テキストモード

CSV

CSVファイルを扱うための標準モジュール。ただし、pandasを使用した方が便利。ただし、AWS Lambdaを使用する際には、csvの方が楽かもしれない。

#モジュールの読み込み

import csv

#CSVファイルの読み込み

with open("data/example.csv") as f:
    reader = csv.reader(f)

#CSVファイルへの書き込み

with open("data/example.csv", "w") as f:
    writer = csv.writer(f)
    #単一行の書き込み
    writer.writerow([0, 1, 2])
    #複数行の一括書き込み
    writer.writerows([[1, 2, 3],[4, 5, 6],[7, 8, 9]])

toml/tomllib

tomlファイルを扱うためモジュール。〜 Python 3.10ではtoml、Python 3.11〜ではtomllibを使用する。

#モジュールの読み込み

import toml
import tomllib

#ファイルの読み込み

#toml
with open(file_path, mode="rb") as f:
    data = tomllib.load(f)
username = data["default"]["name"]

#tomllib
with open(file_path, mode="r") as f:
    data = tomllib.load(f)
username = data["default"]["name"]

#ファイルへの書き込み

#toml
with open(file_path, mode="w") as f:
    data = toml.load(f)
    data["default"]["age"] = 20
    toml.write(data)

#tomllib
with open(file_path, mode="w") as f:
    data = tomllib.load(f)
    data["default"]["age"] = 20
    toml.write(data)

xml.etree.ElementTree

xml.etree.ElementTreeはXMLデータを解析および作成するためのモジュール。

#モジュールの読み込み

import xml.etree.ElementTree as ET

#ファイルを読み込み、Elementオブジェクトを取得する

tree = ET.parse("data/example_data.xml")
root = tree.getroot()

#子ノードを取得する

element = root[0]

#Elementオブジェクトの属性

属性 説明 データ型
element.tag elementのタグ名 str
element.attrib elementの属性名とその値 dict
element.text タグで囲まれた値 str

#Elementオブジェクトのメソッド

メソッド 説明 戻り値の型
element.find("tag_name") 特定のタグ名のElementオブジェクトを一つ返す Elementオブジェクト
element.findall("tag_name") 特定のタグ名のElementオブジェクトをすべて返す (引数にXPath式を限定的にサポートしている) list
element.get("attr_name") タグ内の指定した属性の値を返す str
element.set("new_attr_name", "new_attr_value") タグ内に新しい属性と属性値を追加する -
element.remove("attr_name") タグ内の指定した属性と属性値を削除する -
element.append("tag_name") 新しい子ノードを追加する -

shutil

ファイルやファイルの集まりを操作するためのモジュール。

ファイルのコピー

#モジュールの読み込み

import shutil

#targetディレクトリ直下に"sample.txt"をコピー

shutil.copy("../source/sample.txt", "../target/")

#sample.txtの内容をtest.txtにコピー

shutil.copy("../source/sample.txt", "../target/test.txt")
