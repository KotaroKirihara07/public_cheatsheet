ディレクトリ操作

#メソッド集

メソッド 説明 戻り値の型
os.getcwd() 現在のディレクトリを取得 str
os.path.abspath(path) 絶対パスを取得 str
os.path.relpath(path,起点となるpath) 相対パスを取得 str
os.path.join(path1, path2,..) パスを結合する str
os.path.exists(path) パスの存在を確認 bool
os.makedirs("./new_dir") 新規ディレクトリの作成 -
os.chdir(path) ディレクトリの移動 -
os.rmdir("./test1") ディレクトリの削除 -
os.path.isdir(path) 指定したパスがディレクトリかどうか bool

ファイル操作

#メソッド集 (os)

メソッド 説明 戻り値の型
os.remove("./myfile.txt") ファイルの削除 -
os.listdir(path) ディレクトリ直下のファイル名のリストを取得する list

#メソッド集 (os.path)

メソッド 説明 戻り値の型
os.path.exists("./myfile.txt") ファイルが存在するかどうか調べる bool
os.path.isfile(path) 指定したパスがファイルかどうか bool
os.path.splitext(path) ( 拡張子の手前まで, 拡張子 ) のタプルを取得する tuple
os.path.getatime("./myfile.txt") 最終アクセス時刻を取得する ?
os.path.getmtime("./myfile.txt") 最終更新時刻を取得する ?
os.path.getsize("./myfile.txt") サイズ(バイト数)を返す int
os.path.split(path) ( パスの末尾以外, パスの末尾 ) のタプルを取得する tuple
