regular expression

2025/06/21

このページでは正規表現についての説明を行う。

モジュールの読み込み

#モジュールの読み込み

import re

正規表現パターン

#正規表現パターンを準備

ptn = re.compile(r"正規表現パターン", 検索オプション)

#正規表現パターン

正規表現パターン 説明
XYZ XYZという文字列に一致
[XYZ] X、Y、Zのいずれかの1文字に一致
[^XYZ] X、Y、Z以外のいずれかの1文字に一致
[X-Z] X～Zの範囲の中の1文字に一致
[X|Y|Z] X、Y、Zのいずれかに一致
X* 0文字以上のXと一致
X? 0文字か1文字のXと一致
X+ 1文字以上のXと一致
X{n} Xとn回一致
X{n,} Xとn回以上一致
X{m,n} Xとm～n回一致
^ 行の先頭に一致
$ 行の末尾に一致
. 任意の1文字に一致
\w Unicode単語文字、数字、アンダースコアに一致
\W 文字以外に一致
\d Unicode10進数値に一致
\D 数字以外に一致
\n 改行 (ラインフィードに一致)
\r 復帰 (キャリッジリターンに一致)
\t タブ文字に一致
\s Unicode空白文字に一致
\S 空白以外の文字に一致

#検索オプション (複数の検索オプションを設定する場合は 検索オプション | 検索オプション とする)

検索オプション 説明
re.IGNORECASE 大文字小文字を区別しない
re.MULTILINE 複数行モードの有効化
re.DOTALL 「.」が行末記号を含む任意の文字にマッチ (単一行モード)
re.VERVOSE 空白とコメントの有効化
re.ASCII \w、\W、\b、\B、\d、\D、\s、\SでASCII文字に限定したマッチ
re.LOCALE \w、\Wなどをロケールに従って処理する

Matchオブジェクト

#最初にマッチした文字列1つをMatchオブジェクトとして返す

mtch = ptn.search(str1)

#最初にマッチした文字列1つをMatchオブジェクトとして返す

mtch = ptn.match(str1)

#マッチしたすべての文字列をリストとして返す

mtchs = ptn.findall(str1)

#マッチしたすべての文字列をリストとして返す

mtchs = ptn.finditer(str1)

#属性集

属性 説明 戻り値の型
ptn.pos 検索時に指定された開始位置 int
ptn.endpos 検索時に指定された終了位置 int
ptn.lastindex 最後にマッチしたインデックス int

#メソッド集

メソッド 説明 戻り値の型
ptn.group(N) マッチした文字列を返す (N=0のとき文字列全体、N>=1のときサブマッチ文字列) str
ptn.start() 開始位置を返す int
ptn.end() 終了位置を返す int
ptn.span() (開始位置, 終了位置)のタプルを返す tuple
ptn.sub("new", str1) str1内の正規表現にマッチする箇所を"new"に置き換える str
ptn.split(str1) str1を分割する list
