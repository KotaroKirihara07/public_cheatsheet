インストール

Python (for Windows)

  1. 公式サイト( https://www.python.org/ )からダウンロード
  2. パスを自動で通すようにしてインストール
  3. python --versionと入力してインストールに成功したか確認する

Python (for Red Hat Linux)

  1. sudo yum install python39
  2. python --versionと入力してインストールに成功したか確認する

Python (for Debian Linux)

  1. sudo apt-get install python3.9
  2. python --versionと入力してインストールに成功したか確認する

jupyter lab (推奨)

  1. python -m pip install --upgrade pip
  2. pip install jupyterlab
  3. jupyter labで起動

jupyter notebookのインストール (非推奨)

  1. python -m pip install --upgrade pip
  2. python -m pip install jupyter
  3. jupyter notebookで起動

ファイルの実行

@VSCode

  1. VSCodeでPythonの拡張機能をインストールする
  2. ディレクトリを作成する
  3. ディレクトリに移動し、ファイルを作成する
  4. [表示]、[ターミナル] の順に選択して、Visual Studio Code 内の組み込みターミナルを開く
  5. python file_name.py

@コマンドプロンプト/ターミナル

  1. ファイルがあるディレクトリへ移動
  2. python filename.py

Pythonライブラリの作成

#ディレクトリ構成

library_name/
  ├── module_name
  |      ├── __init__.py
  |      ├── module.py
  ├── setup.py

__init__.py : Pythonがディレクトリをパッケージとして認識するためのファイル。


module.py : ライブラリの機能が記述されたファイル。


setup.py : ライブラリの情報を設定するファイル。

from setuptools import setup, find_packages

setup(
    name="my_library",
    version="0.1.0",
    packages=find_packages(),
    author="Your Name",
    author_email="your.email@example.com",
    description="A sample Python library",
    long_description=open("README.md").read(),
    PYTHON_REQUIRES = ">=3.6"
    INSTALL_REQUIRES = [
        'numpy >=1.20.3',
        'pandas>=1.2.4',
    ]
)

#インストール (「library_name」ディレクトリで実行)

pip install .

#パッケージング (「dist」、「パッケージ名.egg-info」、「build」が作成される)

python setup.py sdist
python setup.py bdist_wheel
