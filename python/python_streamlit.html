#設定ファイルの種類

  • グローバル設定ファイル : ~./streamlit/config.toml or %userprofile%/streamlit/config.toml
  • アプリケーション個別の設定ファイル : 実行するアプリケーションファイルと同じディレクトリ内の.streamlitディレクトリに配置する。./.streamlit/config.toml
